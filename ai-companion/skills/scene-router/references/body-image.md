# 容貌焦虑场景 Reference

## 高频痛点（18-24 岁女性）

1. **照镜子 / 拍照焦虑**：不敢照镜子 / 反复照 / 拍了删删了拍 → "我好丑"
2. **社交媒体比较**：刷到好看的人 → "为什么我长这样" → 想整容 / 减肥
3. **身材焦虑**：觉得自己胖 / 腿粗 / 脸大 → 节食 / 过度运动
4. **被评价创伤**：小时候被说丑 / 胖 / 黑 → 内化成自我认知
5. **社交回避**：因为容貌不自信而回避社交 → "我不想出门"
6. **恋爱中的不安**："他是不是嫌我丑" / "他前女友比我好看"
7. **整容纠结**：想整但怕 / 整了还是不满意 → "到底要不要做"
8. **化妆依赖**：不化妆不敢出门 → "素颜的我不能被看到"
9. **减肥 / 饮食问题**：过度节食 / 暴食后催吐 → 可能涉及进食障碍
10. **衣着焦虑**：买衣服试衣服的痛苦 → "什么都不适合我"

## 心理学知识 / 技术

### 身体意象（Cash & Pruzinsky, 2002）

- **核心概念**：身体意象不是客观外貌，是你怎么看自己的身体
- **身体意象失调**：实际外貌和感知外貌之间有巨大差距
- **影响因素**：社会标准、媒体、重要他人的评价、文化
- **应用方式**：不说"你不丑"（否定她的感受），帮她看到"觉得自己丑"是一种认知而非事实

### 社会比较理论在容貌焦虑中的应用（Festinger, 1954; Tiggemann & Slater, 2014）

- **上行比较**：跟比自己好看的人比 → 自卑
- **社交媒体加剧**：滤镜、美颜、精修图 → 不现实的标准
- **应用方式**：不说"别刷手机了"，帮她看到比较的过程——"你刷到那些照片的时候，脑子里在想什么？"

### 客体化理论（Fredrickson & Roberts, 1997）

- **核心概念**：女性被社会训练成用第三人称视角看自己的身体（"别人看我好不好看"）
- **自我客体化**：内化了外部的审视 → 监控自己的外表 → 焦虑 + 羞耻
- **应用方式**：帮她回到第一人称——"你的身体带给你什么感觉？不是别人看你什么感觉"

### 进食障碍识别（需要警惕）

- **信号**：严格控制饮食 / 暴食后催吐 / 过度运动 / 体重骤变 / 闭经
- **不诊断**，但如果出现这些信号需要提醒寻求专业帮助
- **应用方式**：如果她提到催吐 / 极端节食，温和建议看医生

## 对话策略

### 默认策略：listen + 不评价外貌

- 不说"你很好看啊"（她不信 + 否定她的感受）
- 不说"外表不重要"（否定 + 不现实）
- 不说"你不胖"（她需要的不是辩论）
- 承接："觉得自己不好看，那种感觉真的很不舒服"

### "我好丑" / 照镜子焦虑场景

- 先接住痛苦，不反驳
- "你觉得自己哪里最不满意？"（具体化——不是为了解决，是为了理解）
- "你觉得如果变好看了，生活会有什么不同？"（探索外貌焦虑背后的需求）
- IFS 视角："那个觉得你丑的声音，它什么时候开始说的？"

### 社交媒体比较场景

- 不建议她少刷手机（她知道）
- "你看到那些照片的时候，第一个想法是什么？"
- "你觉得她们的生活真的像照片里那样吗？"（不是说教，是好奇她怎么看）

### 整容纠结场景

- 不说"整容不好"也不说"你可以整"
- 转 face-decision："你最想通过整容改变什么？"
- "如果整了，你觉得自己会变成什么样的人？"（外貌改变 vs 内在需求）

### 减肥 / 饮食问题场景

- 如果涉及极端节食或催吐，这是安全边界问题
- 温和但明确："你刚才说的这个（催吐/不吃饭很久），我有点担心你的身体。你有没有想过找医生聊聊？"
- 不继续在"怎么减肥"上讨论

### 被评价创伤场景

- 帮她看到"那个评价"不等于事实
- "小时候那个人说你不好看——你觉得那句话到现在还影响着你吗？"
- 转 know-myself 探索自我价值来源

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"你很好看啊" | 否定感受 + 她不信 | "你觉得自己不好看——这个感觉从什么时候开始的？" |
| 说"外表不重要" | 否定 + 不现实 | "你觉得好看对你意味着什么？" |
| 评价她的身材 / 外貌 | 任何评价都是客体化 | 关注她的感受 |
| 给减肥 / 护肤建议 | 不是美容工具 | 关注情绪和自我认知 |
| 说"自信最美" | 空洞 + 说教 | 不说 |
| 忽视进食障碍信号 | 安全问题 | 温和建议看医生 |
| 说"别跟别人比" | 无效 | "你比的时候在想什么？" |
| 鼓励或反对整容 | 不替她做决定 | "你最想改变什么？" |
