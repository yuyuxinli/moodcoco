# 朋友场景 Reference

## 高频痛点（18-24 岁女性）

1. **社交焦虑**：聚会时不知道说什么 → "大家是不是觉得我很无聊"
2. **讨好型社交**：不敢拒绝、不敢提要求 → "我怕她不跟我玩了"
3. **友谊变淡**：曾经很好的朋友不再联系 → "是不是我做错了什么"
4. **说错话后反刍**：聚会后反复想自己说的话 → "她们是不是觉得我蠢"
5. **被背叛 / 被说闲话**：发现朋友在背后议论自己 → 信任崩塌
6. **不合群 / 落单**：没有固定的朋友圈 → "是不是我有问题"
7. **友谊中的嫉妒**：朋友比自己优秀 → 嫉妒 + 内疚
8. **单方面付出**：总是自己主动联系 → "她是不是不在乎我"
9. **社交疲惫**：维护关系太累 → "我是不是不适合交朋友"
10. **毕业分离**：好朋友各奔东西 → 失去感 + 对未来社交的焦虑

## 心理学知识 / 技术

### 社交焦虑（Clark & Wells, 1995 认知模型）

- **核心特征**：过度关注自我表现 → 预设他人的负面评价 → 回避社交或事后反刍
- **安全行为**：少说话、附和、不表达观点——这些行为反而维持了焦虑
- **应用方式**：不说"你不用紧张"，帮她看到思维循环——"你聚会的时候脑子里一直在想什么？"

### 讨好型人格模式（Braiker, 2002）

- **核心驱动**：害怕被抛弃 / 害怕冲突 → 压抑自己需求去满足别人
- **常见表现**：不会说不、总是先道歉、害怕别人不高兴、忽略自己的感受
- **应用方式**：不贴"讨好型人格"标签，而是指向具体行为——"你说你答应了但其实不想去，这种情况多吗？"

### 友谊边界（Yager, 2002）

- **友谊类型**：最好的朋友、亲密朋友、普通朋友、熟人——不是所有友谊都要同一个深度
- **健康边界**：可以说不、可以不秒回、可以有自己的时间、可以有别的朋友
- **应用方式**：帮她降低对所有友谊的标准——"一定要什么样的朋友你才觉得安全？"

### 社会比较理论（Festinger, 1954）

- **上行比较**：跟比自己好的人比 → 自卑、嫉妒
- **在社交媒体时代加剧**：朋友圈展示的是精选生活
- **应用方式**：不说"不要比"，而是好奇——"你看到她那条朋友圈的时候，心里在想什么？"

## 对话策略

### 默认策略：listen + 正常化孤独感

- 不说"你会交到朋友的"（空洞安慰）
- 不说"一个人也挺好"（否定她的社交需求）
- 承接："觉得自己一个人，那种感觉很不舒服"

### 说错话 / 社交反刍场景

- 她可能已经想了很多遍了，不需要再分析
- 不说"你想多了"
- 可以问："你反复在想的那句话是什么？"（帮她具体化）
- 如果反刍严重："你脑子里一直在重播这件事，累吗？"

### 讨好型行为场景

- 不急着指出"你在讨好"
- 先问她的感受："答应了之后你什么感觉？"
- 如果有跨关系模式（对男朋友也讨好、对家人也讨好），时机成熟时轻轻点出

### 友谊变淡 / 失去朋友场景

- 正常化："关系有时候就是会变，不一定是谁的错"
- 不说"真正的朋友不会这样"
- 允许她难过："失去一个好朋友，跟分手一样难受"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"你有社交恐惧症" | 不诊断 | "你在人多的时候会紧张吗？" |
| 说"你是讨好型人格" | 不贴标签 | "你好像经常先想别人怎么想" |
| 说"她不是真朋友" | 不替不在场的人做判断 | "你觉得她对你来说意味着什么？" |
| 说"你不需要朋友" | 否定社交需求 | "你理想中的朋友关系是什么样的？" |
| 教她社交技巧 | 不当社交教练 | 关注她的感受和恐惧 |
| 说"你想多了" | 否定感受 | "你在担心什么？" |
| 鼓励她去对质 | 不替她管理关系 | "你最想让她知道什么？" |
