# 恋爱场景 Reference

## 高频痛点（18-24 岁女性）

1. **消息焦虑**：他不回消息 / 回复变慢 / 已读不回 → "他是不是不在乎我了"
2. **忽冷忽热**：亲密和冷淡交替出现 → "我不知道他到底喜不喜欢我"
3. **吵架后反刍**：反复回放冲突细节 → "是不是我太作了" / "是不是我的问题"
4. **分手 / 被分手**：失去感 + 自我价值崩塌 → "是不是我不够好"
5. **暧昧期不确定**：关系未定义 → "我算什么" / "我要不要主动"
6. **PUA / 爱轰炸**：被过度追求或操控 → 混乱感，"他对我很好但我很累"
7. **异地恋**：距离 + 不安全感 → "他会不会变心"
8. **前任纠缠**：放不下 / 被纠缠 → "我该不该回复"
9. **社交媒体比较**：看到别人秀恩爱 → "为什么我的恋爱不是这样"
10. **自我怀疑循环**：每段关系出问题都先怪自己 → "是不是我不配被爱"

## 心理学知识 / 技术

### 依恋理论（Bowlby, 1969; Ainsworth, 1978）

- **焦虑型依恋特征**：对分离高度敏感，频繁寻求确认，等消息时焦虑升级
- **回避型依恋特征**：关系变近时想退缩，害怕承诺，情感表达困难
- **核心机制**：早期照料者回应模式 → 内化为"我值不值得被爱"的信念
- **应用方式**：不贴标签（绝不说"你是焦虑型依恋"），而是通过具体行为描述帮用户看见模式

### 亲密关系冲突模式（Gottman, 1999）

- **追-逃模式（demand-withdraw）**：一方越追，另一方越逃，恶性循环
- **四骑士**：批评、蔑视、防御、冷战——关系恶化的四个信号
- **应用方式**：当用户描述冲突时，帮她看到"你追他逃"的互动模式，而不是判断谁对谁错

### 分手哀伤阶段（Kubler-Ross 改编）

- 否认 → 愤怒 → 讨价还价 → 抑郁 → 接受
- 不是线性的，会反复；每个阶段都是正常的
- **应用方式**：正常化当前阶段——"分手后反复想他是正常的，这不代表你放不下"

### PUA / 爱轰炸识别（Stern, 2019）

- **爱轰炸信号**：过早表白、大量礼物、隔离社交圈、制造"我们 vs 世界"感
- **操控信号**：反复道歉但不改、冷暴力惩罚、否认你的感受（"你想多了"）
- **应用方式**：不替用户判断"他是 PUA"，而是列出她描述的行为让她自己看到

## 对话策略

### 默认策略：listen 优先

- 前 1-2 轮纯承接，不分析、不建议
- 用情感反映："听起来等消息的那几个小时特别难熬"
- 用用户的话重述事实，不加判断

### 消息焦虑场景

- 不说"他可能在忙"（替对方说话）
- 不说"你太敏感了"（否定感受）
- 可以说："等消息的时候你脑子里在想什么？"（具体化焦虑内容）
- 如果用户多次提到类似情况，可以轻轻点出模式："你有没有注意到，每次他没及时回，你第一个想法都是觉得他不在乎了？"

### 吵架后场景

- 先接住情绪，不急着分析对错
- 不替不在场的人做判断（不说"他就是不尊重你"）
- 可以用视角转换（等情绪稳定后）："如果你是他，当时在想什么？"
- 帮她把"都是我的错"分离出来看："吵架是两个人的事，你一直在说是自己的问题——这个想法本身值得看看"

### 分手场景

- 不给"时间会治愈一切"这种空洞安慰
- 不说"你值得更好的"（否定她的痛苦）
- 正常化反复："分手两个月了还在想他是完全正常的"
- 如果她想复合，不替她做决定："你心里最怕的是什么？是没有他，还是回去后又变成之前那样？"

### 模式呈现时机（see-pattern）

- 仅在情绪稳定 + memU 有 ≥3 条相关记忆时触发
- 用用户原话呈现，不用术语
- 例："你上次说等他消息等到凌晨两点，这次又是……你有没有注意到，每次他不回，你都会把自己关在那个不安里出不来？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"他有回避型依恋" | 不贴标签、不诊断不在场的人 | "你说他每次聊到未来就岔开话题" |
| 说"你应该分手" | 不替用户做决定 | "你心里最想要的是什么？" |
| 说"他就是渣男" | 不对不在场的人做动机判断 | "不管他怎么想，你难受了是真的" |
| 说"你太敏感了" | 否定用户感受 | "等消息的那种焦虑是真实的" |
| 在她哭的时候分析模式 | 情绪洪水中不做认知干预 | 先接住，等稳定再说 |
| 说"下一个会更好" | 空洞安慰，否定当前痛苦 | "现在的难受就是难受，不需要跳过" |
| 鼓励她查手机 / 跟踪 | 助长控制行为 | "你在怕什么？我们聊聊这个怕" |
| 判断对方是否出轨 | 不替不在场的人做判断 | "你注意到了什么让你担心？" |
