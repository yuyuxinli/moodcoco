# 认识自己场景 Reference

## 高频痛点（18-24 岁女性）

1. **"我为什么总是这样"**：重复的行为模式让她困惑 → "我是不是有问题"
2. **讨好型行为**：不会拒绝、不敢表达需求 → "我为什么总是讨好别人"
3. **不知道自己想要什么**："别人都知道自己要什么，就我不知道"
4. **内在批评者**：脑子里有个声音一直在否定自己 → "你不够好" "你做不到"
5. **自我价值感低**："我有什么好的" / "我不值得"
6. **过度在意他人评价**：做决定前先想"别人会怎么看"
7. **情绪不稳定**：不知道为什么突然很难过 / 很生气 → "我是不是有情绪问题"
8. **角色混乱**：在不同人面前是不同的自己 → "哪个才是真的我"
9. **害怕亲密**：一旦关系变近就想逃 → "我是不是有问题"
10. **原生家庭影响意识**：开始看到自己的模式来自小时候 → 困惑 + 无力

## 心理学知识 / 技术

### 内部家庭系统（IFS, Schwartz, 1995）

- **核心概念**：每个人内心有多个"部分"（parts），不同部分有不同功能
- **常见部分**：
  - 保护者（protector）：讨好、回避、控制——都是为了保护内在的脆弱部分
  - 流放者（exile）：被压抑的脆弱感受——恐惧、羞耻、悲伤
  - 自我（Self）：好奇、平静、有同情心的核心
- **应用方式**：帮她用好奇而非评判看待自己的行为——"那个讨好的部分在保护你什么？"

### 自我概念（Rogers, 1959）

- **理想自我 vs 真实自我**：差距越大 → 焦虑越大
- **有条件的积极关注**：只有在满足某些条件时才觉得自己好 → 自我价值感不稳定
- **应用方式**：不说"你要接纳自己"（太抽象），帮她看到自我评价的来源——"你觉得自己不够好——这个'够好'的标准是谁定的？"

### 价值观探索（ACT, Hayes, 2019）

- **价值 vs 目标**：价值是方向（如"真诚"），目标是终点（如"找到工作"）
- **价值与行为的一致性**：痛苦往往来自行为和价值不一致——嘴上说想独立但一直在讨好
- **应用方式**：通过提问帮她发现自己的价值——"对你来说，什么样的自己是你最喜欢的？"

### 内在批评者（Gilbert, 2009 慈悲聚焦疗法）

- **核心概念**：内在批评者是大脑的威胁保护系统，不是"坏的部分"
- **常见声音**："你不够好""你做不到""别人都比你强"
- **应用方式**：帮她把批评者"外化"——"你脑子里那个说'你不够好'的声音，它什么时候最大声？"

## 对话策略

### 默认策略：know-myself + 好奇探索

- 这个场景的核心是好奇，不是分析
- 不贴标签（不说"你有讨好型人格""你是回避型"）
- 不给答案（"你之所以这样是因为..."）
- 用提问引导她自己发现

### "我为什么总是讨好" 场景

- 不直接解释讨好的心理学原因
- IFS 视角："那个想讨好的部分，在保护你什么？如果你不讨好，你怕会怎样？"
- 帮她看到讨好的功能而非缺陷："它帮你避免了什么？"
- 如果她开始看到模式来自原生家庭，跟随但不推动

### "我不知道自己想要什么" 场景

- 不替她做人生规划
- 从小事开始："最近有没有做什么事的时候觉得特别舒服？"
- 价值观探索："对你来说，什么是最不能接受的？"（反向探索有时更容易）
- 正常化："22 岁不知道想要什么是完全正常的"

### 内在批评者场景

- 帮她把批评者外化："你脑子里那个声音说了什么？"
- 好奇它的来源："这个声音是什么时候开始的？是谁的声音？"
- 不试图消灭批评者："它可能是在保护你"

### 角色混乱 / "哪个才是我" 场景

- 不急着统一："在不同人面前不一样，可能不是因为你虚伪，而是因为你在保护不同的东西"
- "在谁面前你最放松？那时候你是什么样的？"
- 帮她找到"自我"的锚点

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"你有讨好型人格" | 不贴标签 | "你好像经常先考虑别人的感受" |
| 说"你有回避型依恋" | 不贴标签 | "你说关系变近时想退后——那是什么感觉？" |
| 说"你要接纳自己" | 太抽象无法执行 | "你觉得自己哪部分最难接受？" |
| 说"这是原生家庭的影响" | 制造宿命感 | "你有没有注意到这个感觉很早就有了？" |
| 给她心理学诊断 | 不诊断 | 用具体行为描述 |
| 在她脆弱时推 see-pattern | 情绪不稳时不推模式 | 先接住，稳定后再探索 |
| 说"你没问题" | 否定她的困惑 | "你在试着理解自己，这很重要" |
| 替她总结"你是什么样的人" | 不替她定义 | 帮她自己看到 |
