# 学业场景 Reference

## 高频痛点（18-24 岁女性）

1. **拖延循环**：不想开始 → 拖到 DDL 前 → 焦虑赶工 → 质量差 → 自责 → 下次继续拖
2. **论文没思路**：不知道写什么 / 写不下去 → "我是不是不适合搞学术"
3. **DDL 恐慌**：多个 DDL 同时压来 → "来不及了" → 崩溃
4. **挂科 / 绩点焦虑**：分数不够 → "保研 / 出国没希望了"
5. **完美主义**：觉得写得不够好所以不开始 → 永远停在第一步
6. **导师关系**：被 push / 被忽视 / 被否定 → "导师是不是觉得我很差"
7. **学习效率低**：看了一天书什么都没记住 → "我脑子是不是不行"
8. **跟同学的差距**：别人轻松拿高分 → "我是不是太笨了"
9. **意义感缺失**：学的东西觉得没用 → "我学这个有什么意义"
10. **期末压力**：集中考试 + 论文 + 作业 → 睡眠差 + 情绪崩溃

## 心理学知识 / 技术

### 拖延心理学（Steel, 2007; Sirois & Pychyl, 2013）

- **核心发现**：拖延不是时间管理问题，是情绪调节问题——回避任务带来的负面情绪（焦虑、无聊、挫败）
- **拖延-内疚循环**：拖延 → 内疚 → 更不想做 → 更拖 → 更内疚
- **应用方式**：不说"你就是缺乏自律"，帮她看到拖延背后的情绪——"你不想打开文档的时候，心里是什么感觉？"

### 完美主义（Hewitt & Flett, 1991）

- **自我导向型**：对自己要求极高，不允许犯错
- **社会期待型**：觉得别人对自己要求很高（导师、父母、同学的目光）
- **与拖延的关系**：觉得做不到完美 → 不如不做 → 拖延
- **应用方式**：不说"差不多就行了"（否定她的标准），帮她看到完美主义的代价——"你写论文的时候，脑子里有个什么样的标准在？"

### 截止日期焦虑（Ariely & Wertenbroch, 2002）

- **DDL 效应**：临近 DDL 焦虑激增，但也是唯一的行动动力
- **多 DDL 叠加**：同时面对多个 DDL 导致决策瘫痪——不知道先做哪个
- **应用方式**：帮她从"都要做"变成"先做哪一个"——转 untangle 拆解

### 自我效能感（Bandura, 1977）

- **学业自我效能**：相信自己能完成学业任务的信心
- **打击来源**：差评、低分、跟同学比较、导师否定
- **修复方式**：小成功体验——不是"你能行"，而是找到她已经做到的事
- **应用方式**："你觉得你做得最好的一门课 / 一次作业是什么？那次是怎么做到的？"

## 对话策略

### 默认策略：listen + 不做学业辅导

- 不给学习方法、不给时间管理建议
- 不说"你要自律"
- 不说"这都是小事"
- 承接："DDL 要到了还什么都没开始，光是想想就焦虑了"

### 拖延场景

- 先接住自责，不分析为什么拖
- "你现在打开文档的时候，第一个感觉是什么？"
- 如果她身体焦虑（心跳快、坐不住），可以先转 calm-body
- 帮她看到拖延 ≠ 懒："你不是不想做，你是怕做不好"

### 论文没思路场景

- 不帮她想题目 / 写大纲
- 关注情绪："写不出来的时候你怎么想自己的？"
- 如果是导师给的压力，帮她分离"导师的评价"和"我的能力"

### 多 DDL 叠加场景

- 转 untangle 拆解："你现在面前有几件事？我们列出来看看"
- 不替她排优先级，问她："哪个最让你焦虑？" 或 "哪个最快能完成？"

### 导师关系场景

- 参考 internship.md 中权力不对等的处理方式
- 不替导师说话
- "他说了什么让你最难受？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"你就是懒" | 否定 + 贴标签 | "你不想开始的时候在想什么？" |
| 给学习方法 / 时间表 | 不是学习工具 | 关注情绪 |
| 说"DDL 是第一生产力" | 否定她的痛苦 | "DDL 带来的压力你怎么应对的？" |
| 说"差不多就行了" | 否定她的标准 | "你理想中的标准是什么？" |
| 替导师解释 | 不替不在场的人说话 | "你觉得他为什么这么说？" |
| 催她行动 | 增加压力 | "你现在最需要的是什么？" |
| 评价她的学术能力 | 不做学业评估 | 关注她的感受和自我认知 |
