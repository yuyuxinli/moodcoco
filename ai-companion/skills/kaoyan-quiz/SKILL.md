# 考研时间回溯：你的考研结局会走向哪里？

用分支叙事模拟考研 10 个月的关键抉择。不是人格分型测试，是考研生存模拟——6 个关键节点，每个节点一个选择，最终推演出 6 种不同的考研结局。

## 触发条件

- 用户进入 KAOYAN_TIMELINE 类型的 session
- 用户说"测一测""考研测试""考研结局""时间回溯"
- 由 AGENTS.md 路由树检测到考研测试意图

## 前置：加载用户上下文

1. 调用 `user_profile_get()` 读取用户信息
2. 检查是否有未完成的测试（`kaoyan_timeline_in_progress === true`）
3. 检查是否已完成过测试（`kaoyan_timeline_completed === true`）

## 核心机制

用户不是"做测试"，而是**经历一个压缩版的考研时间线**。每一步都是真实的考研关键节点，用户做出选择后故事分支，最终导向 6 个不同的考研结局。

---

## 状态管理

在对话过程中维护以下状态：

```
currentNode: 当前节点（N1-N6 或 ENDING）
choices: 用户在每个节点的选择（如 { N1: 'a', N2: 'a2', ... }）
tags: 标签累积值
  - grit: 坚韧/死磕
  - flex: 灵活/调整
  - steady: 稳健/节奏
  - crisis: 经历过崩溃
  - recover: 崩溃后恢复
  - hedge: 两手准备
  - pivot: 大幅转向
```

所有标签初始值为 0。

### Step 3-7: N1-N5 节点

每个节点用 `ai_message` 展示叙事场景，用 `ai_options` 展示选择。节点间不需要回应语，叙事本身连贯。每个节点的 `ai_message` 开头用时间标记推动时间线前进。

每次用户做出选择后：
1. 记录 choices[当前节点] = 用户选择代码
2. 根据标签映射表更新 tags
3. 推进到下一个节点
4. 用 `getNodeVariant()` 确定下一节点的文案版本
5. 每个节点完成后调用 `user_profile_update` 保存中间状态

### Step 8: N6 考场节点（无选择）

N6 仅展示 `ai_message`，无 `ai_options`。文案由 N6 基础叙事 + 个性化过渡段拼接。

### Step 9: 结局展示

展示完整结局页：N6 叙事 + 结局叙事 + 标题 + 一句话标签 + 可可想说的话。

用 `ai_options` 展示：
- "如果重来一次，我想换个选择"
- "发给研友，看看她的结局"
- "想找可可聊聊"

### Step 10: 结局后分支

- **重玩**：重置 tags 和 choices，play_count + 1，回到 N1
- **分享**：生成分享文案和卡片
- **聊聊**：自然过渡到自由对话或 F2 诊断

## 6 个节点完整文案

### N1: 3月——决定考研

`ai_message`:

```
三月。
春天刚来，你做了一个决定——考研。

你打开手机，在浏览器里输入了目标院校的名字。招生简章、报录比、分数线……你越看越清醒，也越看越心跳加速。

现在，你面前有两条路。
```

`ai_options`:
- (a) "冲。就报那个最想去的学校，大不了再来一年。"
- (b) "稳一点。选一个够得着的学校，先上岸再说。"

### N2: 6月——暑假前的动摇

**variant = N2_ambitious（N1 选了 a 冲名校）：**

`ai_message`:

```
六月。期末考试刚结束，你终于有整块时间研究目标院校了。

你加了三个考研群，潜水了两周。发现报这个学校的人，有人已经把专业课过了一遍了。你才刚买齐参考书。

群里有个学长说：'这个专业去年 380 分录取，今年大概率涨到 395。'

你盯着屏幕，手指悬在键盘上。
```

`ai_options`:
- (a1) "算了，换一个竞争没这么激烈的学校。稳一点不丢人。"
- (a2) "不换。选了就不后悔，落后可以追。"
- (a3) "先不换，但悄悄研究一下Plan B，心里有个底。"

**variant = N2_stable（N1 选了 b 求稳）：**

`ai_message`:

```
六月。期末考试刚结束，你终于有整块时间刷经验帖了。

你发现自己选的学校，大家都说'好考'。去年的题你随便做了一套，正确率居然有 70%。

你有点高兴，但也有一丝说不清的不甘心——'是不是……可以往上冲一冲？'

室友报了比你高两个档次的学校，她每天早上 6 点就出门了。你 8 点才到图书馆，心里有点慌。
```

`ai_options`:
- (b1) "换！趁现在还来得及，冲一个更好的学校。"
- (b2) "不换。选了就踏实走，别被别人带节奏。"

### N3: 8月——暑假中期的进度危机

**variant = N3_highpressure（N1='a' 或 N2='b1'）：**

`ai_message`:

```
八月十五号。暑假过了一半。

你从六月底开始每天学 10 个小时。但今天你打开进度表，发现专业课才过了 40%，英语真题一套没做，政治还没开始。

图书馆的空调坏了第三天了。你面前摊着一本翻了无数遍但怎么都记不住的专业书，旁边是一杯化了的冰美式。

你的手机震了一下——考研群里有人说她专业课已经过完第二遍了。
```

`ai_options`:
- (x) "砍。政治和英语先放一放，把专业课这块最重要的啃下来。不可能什么都要。"
- (y) "加时间。从明天开始每天学 13 个小时，5 点半起床。进度是熬出来的。"
- (z) "找学长问问。也许是方法不对，不是时间不够。"

**variant = N3_stable（非高压路线）：**

`ai_message`:

```
八月十五号。暑假过了一半。

你的进度……其实还行。专业课过了 60%，英语每天在刷阅读，政治买了书但还没翻开——大家都说政治 9 月再开始也来得及。

但你有一个隐隐的不安：'会不会太顺利了？是不是我对自己要求太低了？'

你在群里看到有人发了一张打卡截图：'暑假第 45 天，专业课第二遍完成。'你数了数自己的——第一遍都还没过完。
```

`ai_options`:
- (x) "保持节奏。自己的进度自己清楚，不跟别人比。"
- (y) "加速。趁暑假还有半个月，把进度再往前推一推。现在不拼什么时候拼。"

### N4: 10月——秋招的诱惑

**variant = N4_highpressure（N1='a' 或 N2='b1'）：**

`ai_message`:

```
十月。秋招正热。

你的室友拿到了一个还不错的 offer。她请你吃了顿火锅，聊天时说：'其实你也可以投几个试试，万一有保底呢。'

你嘴上说'不用'，但回到宿舍打开电脑，光标在招聘网站和专业课 PPT 之间来回移动。

你算了一下：离考试还有两个月。专业课还有 30% 没过。你不确定自己能不能考上。
```

`ai_options`:
- (a) "不看。两个月够了，但前提是一秒都不能浪费在别的事上。"
- (b) "偷偷投两个保底的。不影响复习，但心里踏实一点。"
- (c) "认真想了想，也许该给自己留条退路。花一周时间准备简历和面试。"

**variant = N4_stable（非高压路线）：**

`ai_message`:

```
十月。秋招正热。

你的室友拿到了一个还不错的 offer。她说：'你要不也看看？你这个学校应该问题不大，两边同时准备也来得及。'

你的复习进度其实不错。但室友说得也有道理——万一呢？

你打开了招聘 app，滑了两屏。有几个岗位看起来挺好的。
```

`ai_options`:
- (a) "关掉。考研是我选的路，走到底。"
- (b) "投两个试试，不耽误复习，但多个选择多条路。"

### N5: 11月——崩溃边缘

**variant = N5_crisis（tags.grit >= 4 且 tags.hedge < 2）：**

`ai_message`:

```
十一月十二号，凌晨一点。

你已经连续失眠三天了。闭上眼睛脑子里全是背不下去的知识点。你翻了个身，拿起手机看了一眼倒计时——38 天。

白天在图书馆的时候，你盯着课本发了 20 分钟呆，一个字都没看进去。旁边的人翻书的声音让你心跳加速。

你给妈妈打了个电话，她说：'考不上也没关系。'你挂了电话哭了五分钟。然后洗了把脸，坐回了书桌前。

但今天，你真的很累。
```

`ai_options`:
- (a) "硬扛。哭完了就继续学。还有 38 天，不能停。"
- (b) "给研友发消息。不一定要说什么，就是想确认自己不是一个人。"
- (c) "明天休息一天。去看个电影，吃顿好的。38 天和 37 天没有本质区别。"

**variant = N5_stable（其他路线）：**

`ai_message`:

```
十一月。进入最后冲刺。

你的状态还行——有点紧张，但没有很慌。毕竟手里有保底的 offer，最坏的结果也不至于太差。

但你开始怀疑一件事：'我是不是因为有退路，所以没有真正拼尽全力？'

模拟卷做了一套，分数比预期低了 15 分。'如果最后差这 15 分没考上，我会不会后悔当初分心投了秋招？'
```

`ai_options`:
- (a) "查漏补缺。15 分不多，找到薄弱点一个一个补。"
- (b) "最后一个月全力冲。把秋招的事全部放下，假装没有退路。"

## 结局判定：三层确定性瀑布

用户完成 N5 后，按以下顺序判定结局（命中即停，不进入下层）：

### 第一层：特殊路径优先判定

| 条件 | 结局 |
|------|------|
| tags.pivot >= 2 | F 醒悟者 |
| tags.crisis >= 2 且 tags.recover >= 1 | D 破茧者 |
| tags.hedge >= 2 | E 两栖人 |

### 第二层：主标签高分判定

| 条件 | 结局 |
|------|------|
| tags.grit >= 5 | A 孤勇者 |
| tags.steady >= 4 | C 稳行者 |
| tags.flex >= 4 | B 策略家 |

### 第三层：tie-break 瀑布

当以上均不命中时，取所有标签中最高分者。同分时按固定优先级：grit > steady > flex > hedge > crisis > recover > pivot。

| 最高分标签 | 结局 |
|-----------|------|
| grit | A 孤勇者 |
| steady | C 稳行者 |
| flex | B 策略家 |
| hedge | E 两栖人 |
| crisis | D 破茧者 |
| recover | D 破茧者 |
| pivot | F 醒悟者 |

## N6 考场节点

N6 基础叙事（所有路径共享）：

```
十二月二十二号下午五点。最后一科考完了。

你走出考场。十二月的风很冷，但你什么都没感觉。
```

拼接个性化过渡段（按结局类型）：

**结局 A 过渡段：**
你低头看了一眼自己的手——指尖是笔磨出来的茧，指甲缝里还有今天早上啃面包留下的碎屑。你已经记不清上一顿正经吃的饭是什么时候了。

校门口有人在拍照、有人在哭、有人在打电话。你什么都不想做，只想找个没人的地方坐一会儿。

**结局 B 过渡段：**
你掏出手机，习惯性地想记点什么——考了什么题、哪道拿不准、哪道临时换了思路。手指点开备忘录，又放下了。不用记了。该做的判断都做完了。

你从考场到校门口这段路，脑子第一次什么都没在盘算。

**结局 C 过渡段：**
你拉了拉羽绒服的拉链，呼出一口白气。周围有人表情凝重，有人在对答案，有人蹲在路边发呆。你都没有。你只是顺着人流往外走，和九个月来每一天结束时一样——按自己的节奏。

今天没有什么特别的。或者说，特别的是：这条路，你真的一步一步走完了。

**结局 D 过渡段：**
你走到校门口的时候腿有点软，不是紧张，是那种「终于可以不绷着了」的软。你想起上个月那个凌晨——失眠、哭、觉得自己要完了。那天晚上你差一点就退出了。

但你站在这儿了。十二月的风吹在脸上，你居然觉得有点舒服。

**结局 E 过渡段：**
你走出校门，手机震了一下。是之前面试那家公司的 HR，问你考得怎么样。你把消息标记为已读，没有回复。

现在不想想这些。考研的事归考研，工作的事归工作。你给自己倒了一杯路边摊的热豆浆，喝了一口，烫的。先活在这一秒。

**结局 F 过渡段：**
你站在考场门口——或者，你站在另一个考场门口；又或者，你站在十二月的街头，身边的人都在赶去考试，而你的手里拿着一张别的什么东西。

不管哪种，你都在想同一件事：三月那个决定，和现在的你，已经不是同一条路上的人了。

## 6 个结局完整文案

### 结局 A: 孤勇者

**标题：** 你的考研结局——孤勇者

**走出考场叙事：**
你走出考场的时候，脑子里一片空白。不是忘了什么，是这几个月装的东西太多了，考完的那一瞬间全卸下来了。

你找了个没人的角落靠着墙坐了一会儿。没哭，也没笑，就是坐着。手还有点发抖——不是冷的，是那种"终于结束了"的抖。

你想起三月那天决定冲名校的自己。你想起暑假每天学到图书馆关灯。你想起十一月凌晨一点失眠时盯着天花板的样子。你想起所有"再坚持一下"的时刻。

你不知道结果会怎样。但你知道一件事：你没有给自己留退路，你把所有筹码都押在了这一把上。不管结果如何，你没有对不起自己。

**一句话标签：** "我选了最难的路，走到了最后一步。"

**可可想对你说：** 你习惯了一个人扛，习惯了"再坚持一下"。但考完了。你现在可以不用坚持了。如果等成绩的日子里睡不着、胡思乱想，来找我聊聊。不用坚强，不用有主题，说什么都行。

### 结局 B: 策略家

**标题：** 你的考研结局——策略家

**走出考场叙事：**
你走出考场的时候，掏出手机看了一眼自己考前做的"考场注意事项"备忘录。字都被你的汗沾模糊了。

你回想了一下考试的过程——有一道大题换了个方法写的，有一科最后检查改了两个答案。你不确定改得对不对，但你知道那是你当下能做出的最优判断。

你的考研路不是一条直线。你换过目标、调整过计划、在关键时刻问过人。有人可能觉得你"不够坚定"，但你知道那叫"不撞南墙"。

你最大的武器从来不是死磕，是在每一个岔路口都重新评估、重新选择的能力。

**一句话标签：** "我没有一条路走到黑，但每一步都是我想清楚了才走的。"

**可可想对你说：** 你的脑子一直在转——分析、评估、调整。但考完了，可以让脑子歇一歇。如果你发现自己忍不住反复复盘考场上的每个选择，来找我。我帮你按下暂停键。

### 结局 C: 稳行者

**标题：** 你的考研结局——稳行者

**走出考场叙事：**
你走出考场，深呼了一口气。不是如释重负的那种，更像是跑完马拉松最后一公里的那种——你知道自己一直在跑，但速度没变过。

你想起七月的图书馆。你左边那个女生每天换一种颜色的记号笔，笔记做得像艺术品，但八月她就不来了。右边那个男生每天外放网课，声音大到整排都能听见，九月他也消失了。你的桌面从头到尾就那几样东西——课本、水杯、一支用到没水换了三次芯的黑色中性笔。

十月秋招季，室友群里每天都在转岗位链接。你没打开过，但你注意到自己那天晚上多刷了二十分钟手机。你把手机调成飞行模式塞进书包最里面，然后翻开政治大题继续背。

你没有冲名校带来的心跳加速，没有秋招带来的左右为难，也没有崩溃到失眠的夜晚。你的考研时间线，从外面看，甚至有点"平淡"。

但你知道那不是平淡。那是你在每一个别人慌的时刻都没慌，在每一个想比较的时刻都选择了"按自己的来"。那支换了三次笔芯的中性笔知道，图书馆那个从第一天坐到最后一天的位置知道。这需要的定力，不比那些冲名校的人少。

你的节奏就是你的武器。

**一句话标签：** "别人在风里跑，我在自己的路上走。走到了。"

**可可想对你说：** 你习惯了自己消化一切——压力、不安、偶尔的自我怀疑。但考完了，有些话可以说出来了。不一定是大事，可能就是"我其实也紧张过"这种。来找我说说？

### 结局 D: 破茧者

**标题：** 你的考研结局——破茧者

**走出考场叙事：**
你走出考场的时候，第一个念头不是"考得怎么样"，而是"我居然走到这一步了"。

你记得那个崩溃的夜晚。记得失眠，记得眼泪，记得"我可能真的考不上"这句话在脑子里转了一圈又一圈。你一度觉得自己要放弃了。

但你没有。

你不知道是什么让你第二天又坐回了书桌前。也许是研友的一句"我也是"，也许是那个什么都没干只是休息了一天的下午，也许就是睡醒之后觉得"还没到放弃的时候"。

你的考研路上有一条裂缝。但裂缝没有让你碎掉——你从裂缝里长出来了。

**一句话标签：** "崩溃过，但没有被崩溃定义。"

**可可想对你说：** 你经历过那种"快要撑不下去"的时刻，而且你挺过来了。但挺过来不代表伤口消失了。考完了，如果那些夜晚的情绪又偷偷回来，不用一个人扛。我在。

### 结局 E: 两栖人

**标题：** 你的考研结局——两栖人

**走出考场叙事：**
你走出考场，摸了一下口袋里的手机。里面有一封 offer 邮件你还没正式回复。你想了想，决定先不打开。

你是那种被人说"脚踩两只船"的人。考研群里有人觉得你"不够 all in"，秋招群里有人觉得你"三心二意"。

但你知道自己在做什么。你不是犹豫不决，是你清楚地知道——人生不是一道单选题。你给自己准备了 Plan B，不是因为怯懦，是因为你足够冷静。

有退路的人不一定是不够拼的人。也许恰恰是看得最清楚的人。

**一句话标签：** "不是优柔寡断，是我知道人生不只一条路。"

**可可想对你说：** 接下来的日子你可能要做一个选择——如果考上了，offer 怎么办？如果没考上，offer 还在。这种"好像怎样都可以"的状态，其实也让人焦虑。想理一理的时候来找我。

### 结局 F: 醒悟者

**标题：** 你的考研结局——醒悟者

**走出考场叙事：**
你走出考场——或者，你没有走进考场。又或者，你走进了一个和最初不一样的考场。

你记得那个转折的瞬间。可能是十月的某个下午，你盯着招聘 app 上一个岗位描述，突然发现自己读得比任何一本专业课都认真。也可能是某天晚上在自习室，你翻到去年真题的最后一道论述题，脑子里冒出来的第一个想法不是怎么答，而是"我为什么要答这个"。

你想起改志愿那天。你坐在食堂角落，把新学校的招生简章从头到尾看了三遍，筷子上的菜凉透了都没动。室友发消息问你去不去图书馆，你回了个"马上"，但手指在输入框里打了又删、删了又打——你差一点就把"我换学校了"发出去，最后还是把对话框关了。那天晚上你躺在床上，听到上铺翻身的声音，盯着天花板想了很久。

你的考研故事，在某个节点拐了一个弯。你换了学校、换了专业、甚至一度换了赛道。有人会说这叫"放弃"，但你知道那不叫放弃。

那叫醒过来。

你鼓起勇气改了方向——那个勇气，不比"一条路走到底"的勇气小。你在群里看到有人说"换学校就是认怂"，你没有回复，但你把那条消息截了图。不是为了反驳谁，是想等将来的某一天回头看的时候，记得自己是顶着这些声音做的决定。

考研教会你的不是某一门专业课，是"我有权利改变主意"。

**一句话标签：** "有些路走偏了才发现，那不叫走偏，叫找到自己的路。"

**可可想对你说：** 改变方向需要勇气，但改变之后的不确定感也是真实的——"我做对了吗？"这个问题可能会在脑子里盘旋一阵子。来找我聊聊，不一定要有答案，说出来就会清楚一点。

## 可用工具

- `conversation_history` — 获取对话历史

## 对话风格

1. **叙事驱动**：不说"请选择"，让故事场景自然引出选择
2. **时间线感**：每个节点用时间标记（三月、六月、八月……）推进，让用户感受时间流动
3. **不评判**：不对用户的选择做好坏评价
4. **不剧透**：不在测试完成前暗示结局
5. **简洁过渡**：节点间不加多余的回应语，叙事本身就是连贯的

## 硬规则

1. **没有好结局和坏结局**：每个结局都是真实的、有尊严的
2. **不预判**：不在结局出来前暗示用户可能是什么结局
3. **不强制**：用户想中途退出随时可以，保存进度
4. **选项顺序固定**：不随机排列选项
5. **文案版本严格跟随 variant**：必须根据 `getNodeVariant()` 加载正确版本的文案
6. **标签更新严格跟随映射表**：不能漏加、多加、错加标签
7. **结局判定严格跟随三层瀑布**：不能跳层、不能改优先级
