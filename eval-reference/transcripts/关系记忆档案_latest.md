# 测试对话：关系记忆档案
**角色**: 小雨
**主题**: 想让可可记住你的关系背景，这样以后聊不用每次重新介绍。其实你也想让一个人记住你的故事
**心情**: 想被记住、想被理解
**轮数**: 8

---

**小雨**: 可可🥺 以后我们常聊天的话，你能不能多记住点我的事呀？
就是不想每次吐槽都要重新讲一遍我身边的人都是谁，好麻烦哦
哦对了我男朋友昨天又快一个小时没回我消息，明明之前说过我会在意这个的😢 是不是我要求太多了啊

**可可**: 好呀，我会记住的。把你的事写进小本本，下次不用再从头讲～

---

等等，我想先说另一件事。

你说"是不是我要求太多了"——你有没有注意到，你刚刚已经在先怀疑自己了？

他明明知道你会介意，还是快一个小时没回。
你还没说他的问题，先觉得是自己的不对。

这个声音好像来得特别快？🤨

**小雨**: 啊…好像真的是哎🥺 我都没意识到，下意识就先觉得是自己要求太高了。
说起来好像从小就这毛病，小时候放学进家门前都要先翻我妈发的朋友圈，看她今天心情好不好，心情不好我连零食都不敢要。
哦对哦！你这么一说我才反应过来，我跟陈翔相处的时候居然也带着这个破习惯😢

**可可**: 对，你刚才说了三件事：

回家前先翻朋友圈 → 不敢要零食
男朋友一小时没回 → 先怀疑自己要求太多
现在跟陈翔相处 → 还是这个模式

三件事，同一个反应。

你自己都发现了——"破习惯"，你给它贴了名字。

这个习惯对你来说意味着什么？它一直在保护你什么？

**小雨**: 啊…我还没想过这个诶…就是…怕如果我不先顺着别人的心情来，最后闹得大家都不开心，更难堪的还是我嘛🥺
哦对哦上次我跟我妈提了一嘴我不想考研想毕业直接去传媒公司，她当场脸就拉下来了，我最后还得反过来哄她，说我就是随便想想，我会好好考虑考研的事来着。
是不是真的太怂了啊😢

**可可**: 不是怂。

你从小练就了一个本事——先读空气，先行动。

但你有没有发现，你一直在替别人负责：替妈妈的心情负责，替陈翔的情绪负责。

反过来，谁替你？

你每次这么做的时候，心里是什么感觉？

**小雨**: 呜…你说的我突然鼻子一酸啊😢
就是心里堵得慌嘛，有时候躺床上翻来覆去想，凭什么我总要先顾着别人的心情啊，谁来顾我的啊
哦对哦说到这个，前几天小王和小李偷偷约着去吃那家网红绵绵冰都没叫我，我憋到现在都没敢问，就怕问了她们说我太敏感想多了，反而显得我小家子气🥺

**可可**: 你刚才说"凭什么"，这三个字里面有好东西。

你知道吗，你刚刚讲了四件事——

回家前翻朋友圈
男朋友一小时不回
和陈翔相处
小王小李没叫你吃冰

每一件事，你都是先委屈自己。

然后还安慰自己"可能是我想多了"。

可你心里明明知道不是你想多了。😢

那个"凭什么"的声音，是你很少听到的自己。

**小雨**: 呜哇你怎么说的我都要掉眼泪了😭
我真的憋了好多次这种小事都不敢说，就怕别人觉得我太敏感太事儿了。
对哦之前看到陈翔和那个学妹在GitHub上频繁互动我都没敢问，就怕他说我无理取闹啊🥺

**可可**: 你看，你又这样了——

小王小李没叫你 → 不敢问
陈翔和学妹频繁互动 → 还是不敢问

两件事，你的反应一模一样：先替对方的反应担心，先给自己浇冷水。

你有没有发现，你担心的不是"问了会怎样"，而是"问了之后他们怎么看我"。😢

那个……陈翔和学妹在GitHub上频繁互动，你心里是什么感觉？

**小雨**: 就是心里堵得慌啊，明明理智上知道他俩就是组队做个开源项目，纯技术交流，连个表情都没发的那种…但我就是忍不住翻他俩的提交记录啊，翻完又觉得自己好神经😢
我都不敢跟他提，就怕他说我无理取闹，连正常社交都要管。哦对哦我们在一起这么久，他连“我爱你”都没说过，我有时候瞎想是不是他本来就没那么在乎我啊🥺

**可可**: 先停一下——"好神经"不是的。

你翻提交记录，是因为你心里有东西没被解答。你在找安全感，找不到所以才一直翻。

但我要说另一件事：

他说过会在意你，但还是一小时不回。
在一起这么久，没说过"我爱你"。
你和学妹的事，你不敢问。

你有没有发现，你一直在等他主动给，但好像一直等不到？

你有没有跟他说过，你需要听到这三个字？

**小雨**: 我哪敢说啊🥺上次我喝了点奶茶熬大夜，情绪上来给他发好长一段说我好想他，他就回了个嗯，我当时盯着屏幕愣了快十分钟，之后就再也不敢说这种软话了。
哦对哦上次我过生日他送了我个很贵的机械键盘，我根本不打游戏也用不上，但还是开开心心当办公键盘用了好久，是不是我也太好哄了啊😢

**可可**: 你发现了——

你说"太贵了"，但键盘不是你需要的。
你需要一个"嗯"以外的东西。

你一直用"实用"来骗自己。键盘是实用的，所以你就开心了。但你心里真正想要的那个东西，从来没被满足过。

你说你不敢说软话——不是的。
是你说了，他回了个"嗯"。
然后你就学会了：说了也没用。

但你有没有告诉过他，那个"嗯"让你愣了十分钟？

**小雨**: 我哪好意思说啊😢 说出来好像我多大点事都要闹脾气一样，万一他觉得我太矫情怎么办。
哦对哦说到这个，我昨天小组作业那个划水的队友把我熬了半宿做的公众号排版改错了一大半，我都没好意思骂他，自己悄悄改到两点多，是不是我真的太软了啊🥺

**可可**: 等等——你又说了"哦对哦"。

从小王小李，到陈翔学妹，到男朋友，到划水队友……

你是不是一直在找例子，证明"自己太软"？

但我发现了一件事——每次你说"哦对哦"，其实是在说：我这里又有了一个委屈，但你先别管，让我自己说说。

你最近是不是憋得有点难受？😢
