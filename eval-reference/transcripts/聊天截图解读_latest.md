# 测试对话：聊天截图解读
**角色**: 小雨
**主题**: 发来和男朋友的一段对话文字让可可分析，其实你心里已经有判断了只是想找人确认
**心情**: 不安、需要外部验证
**轮数**: 8

---

**小雨**: 🥺可可你快帮我分析分析嘛

**可可**: 你不是小心眼。

你熬到两点手都冻红了，结果他就回一句"嗯辛苦了早点睡"——你说他看完这话什么感觉？就觉得被接住了吗？

然后你去翻GitHub，发现他跟学妹聊二十多条。。

我不是说你应该生气或者怎么样，但你自己心里清楚：你要的不是一句"辛苦了"，是你发那条的时候，希望他多问两句、多关心一点。🫧

你怀疑自己，可能是他没给你那个"被在乎"的感觉，所以你会往自己身上找原因。

但你现在最难受的，真的只是因为那二十条消息吗？还是因为——**你觉得他对你没以前上心了？**

**小雨**: 呜你怎么一下子就说中了啊😭

哦对哦，我上周发的糖水铺动态他到现在都没点赞，刚才还看到他给那个学妹拍的日落点赞，是不是我平时说的都是些鸡毛蒜皮的小事，他觉得烦啊🥺

**可可**: 你刚才说了三件事——

字数不一样，点赞不一样，然后你得出的结论是：**是不是我的问题。**

可是你想啊，糖水铺是你觉得好吃才发的对吧？那是你的生活，不是鸡毛蒜皮。

他给学妹的日落点赞，是因为学妹拍得多好，还是因为——他就是想给人家点？

我不是说他一定怎么样。我是说，你已经在心里比了一圈，然后每次比完都把责任揽到自己身上。

你有没有发现，你好像习惯了先怀疑自己？

**小雨**: 呜…你说的好像真的对哦😢
我就是下意识会先想是不是我太啰嗦了，说的都是没营养的小事他才不想回的。
哦对哦刚才小王和小李下课直接约着去买热奶宝了，都没问我去不去，我今天怎么什么破事都赶一块啊😭

陈翔回你几个字，你怀疑自己。
小王小李没叫你，你也没说"他们可能忘了"或者"可能有原因"。

你直接又想到"是不是我怎么了"。

男朋友、室友，三件事不一样的对象，但你的反应是一样的——**先觉得自己有问题。**

这不是巧合吧？

**小雨**: 呜…好像真的是这样🥺

对哦我妈今早还打电话催我考研，说我一天天净想些没用的，我都没敢跟她说我不想考想直接去传媒公司😭

**可可**: 今天这么多事一起砸过来，你肯定很难受。

谢谢你愿意说出来——你自己把小时候的事都连上了，这不容易。🫧

你现在想继续聊，还是想先缓一缓？

**小雨**: 我不想缓嘛，憋在心里好久了都🥺
对了可可你说我要不要直接去问他啊？就是那个GitHub的事，我怕问了他觉得我没事找事，可憋着我真的好难受啊。
哦对哦小晴上周说这周末要带她新对象来我们学校这边玩，我都在想要不要叫陈翔一起，到时候他又半天憋不出一句话多尴尬啊。

但我发现你现在最难受的，好像不是“要不要问”“要不要叫”这个问题。

是你想说但不敢说，想被看见但感觉他没看见你。

你有没有想过——你憋着难受，和你开口问他，哪个更破坏你们的关系？

**小雨**: 我也不知道啊…就怕问了他觉得我小心眼，还偷偷翻他账号，反而显得我没事找事😢

但你想啊，不是你天然缺安全感。是你发那条消息的时候，你在乎他怎么回，结果他就回了一句。你会在意，是因为你在乎。

让你睡到三点的，不是那二十条消息。
是你在想——**为什么他跟我说话就没这么多，但跟别人有。**

这个感觉才是真的。那个"缺安全感"的标签，反而让你觉得这是我的问题。

可问题是——他回你五个字的时候，你真的被接住了吗？

**小雨**: 😭对啊我当时还特意拍了冻红的手的照片一起发的，他连提都没提一句。

是不是我要求真的太高了啊🥺

**可可**: 你有没有注意到，你今天已经第三次问"是不是我怎么了"。

——是不是我小心眼，是不是我太敏感，是不是我要求太高。

你拍那张冻红的手的照片发给他，不是想让他帮你解决小组作业，就是想让他看见你。

你说他连提都没提。

然后你现在问我——我要求真的很高吗？

小雨，你刚才自己说的：就乃至一句"快捂捂"就够了。

这要求高吗？

**小雨**: 呜哇你这么说我更想哭了😭
真的一点都不高啊…他之前明明还会记得给我买热奶茶的啊，怎么现在就变成这样了嘛
