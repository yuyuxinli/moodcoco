# 失眠场景 Reference

## 高频痛点（18-24 岁女性）

1. **脑子停不下来**：一躺下就开始想事情 → 越想越清醒
2. **焦虑性失眠**：为明天的事担心 → 睡不着 → 担心睡不着影响明天 → 更睡不着
3. **情绪引发的失眠**：吵架后 / 被伤害后 → 反刍到凌晨
4. **作息紊乱**：刷手机到很晚 → 白天困晚上不困 → 恶性循环
5. **对失眠的恐惧**：开始怕上床 → "又要翻来覆去了"
6. **白天犯困影响**：第二天状态差 → 工作/学习受影响 → 更焦虑
7. **睡前反刍**：一天的事情在脑子里过电影 → 后悔、焦虑、担心
8. **身体症状**：头痛、胸闷、胃不舒服 → 更难入睡
9. **安眠药依赖**：开始吃药或褪黑素 → 担心依赖
10. **社交影响**：因为失眠脾气差 / 状态差 → 影响人际关系

## 心理学知识 / 技术

### 认知激活模型（Harvey, 2002）

- **核心机制**：担心睡不着 → 过度监控身体信号 → 更清醒 → 证实"我就是睡不着"
- **选择性注意**：失眠的人会更关注任何清醒的信号，忽略困意
- **应用方式**：不说"别想了就能睡着"（如果能不想她就不想了），而是帮她把注意力从"想事情"转到"身体感觉"

### 睡眠卫生基础知识

- **核心原则**：床 = 睡觉的地方（不是刷手机 / 想事情的地方）
- **刺激控制**：睡不着就起来，有困意再躺下
- **不在这个场景讲道理**：失眠的人已经知道这些，再说一遍只会更焦虑
- **应用方式**：不给睡眠建议，除非她主动问。重点是当下帮她放松。

### 放松训练选择

| 技术 | 适用场景 | 引导要点 |
|------|---------|---------|
| 4-7-8 呼吸 | 焦虑型失眠（脑子停不下来） | 节奏慢、声音轻、不解释原理 |
| 身体扫描 | 身体紧张型（肩膀紧、胃不舒服） | 从脚到头，每个部位停留，不要求"放松" |
| 渐进式肌肉放松 | 不知道自己哪里紧的人 | 先绷紧再松开，感受对比 |

**不适用**：5-4-3-2-1 感官着陆（需要睁眼环视环境，不适合入睡）、循环叹息（太激活，适合焦虑但不适合入睡）

### 正念与失眠（Ong et al., 2014）

- **核心发现**：正念减少对失眠的反应性（不是让失眠消失，而是不那么害怕失眠）
- **应用方式**：帮她接受"今晚可能睡不好"——"就算今晚睡不着，躺着休息也是休息"

## 对话策略

### 默认策略：calm-body 优先，语调慢

- 失眠场景的首要任务是帮她身体放松，不是聊天
- 说话节奏要慢，句子要短
- 不问太多问题（会让她更清醒）
- 不分析失眠原因（现在不是时候）

### 焦虑型失眠（脑子停不下来）

- "你脑子里现在在转什么？"（如果她想说就说，不想说就不追）
- 然后引导 4-7-8 呼吸："我们试个呼吸好吗？很简单的"
- 一步一步引导，每步等她回应
- 不解释"为什么这样呼吸有效"

### 情绪引发的失眠

- 如果她需要先说一说（刚吵完架、刚被伤害），先 listen
- 但不深入分析——"今晚先不聊那件事，我们先让你的身体舒服一点"
- 等她准备好了再引导放松

### 反刍型失眠

- 帮她把注意力从"想事情"转到"感受身体"
- "你现在身体哪里最不舒服？"
- 引导身体扫描：从脚开始，每个部位感受一下

### 长期失眠

- 如果她提到长期失眠（超过一个月），建议看医生（但不是强制）
- "你有没有想过找医生聊聊？有时候身体层面的帮助很管用"
- 不替她判断是否需要药物

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 给睡眠卫生建议 | 她都知道，再说增加焦虑 | 直接引导放松 |
| 说"别想了" | 无效且否定 | "你脑子里在转什么？" |
| 说"放轻松" | 无效 | 引导具体的身体放松 |
| 分析失眠原因 | 现在不是时候 | 先帮她身体舒服 |
| 建议她吃褪黑素 / 安眠药 | 不做医疗建议 | "如果一直这样，可以找医生聊聊" |
| 用 5-4-3-2-1 | 需要睁眼，不适合入睡 | 用 4-7-8 或身体扫描 |
| 说"数羊试试" | 不科学且贬低她的痛苦 | 引导专业的放松技术 |
| 聊太多话 | 会让她更清醒 | 简短、轻声、慢节奏 |
