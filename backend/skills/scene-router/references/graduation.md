# 毕业场景 Reference

## 高频痛点（18-24 岁女性）

1. **未来不确定性**："毕业了然后呢" → 不知道自己想做什么
2. **身份转换焦虑**：从学生变成社会人 → "我还没准备好"
3. **毕业论文 / 答辩**：写不出来 / 怕被问倒 → DDL 恐慌
4. **分离焦虑**：要跟室友 / 朋友 / 校园告别 → "以后见不到了怎么办"
5. **比较焦虑**：同学各有去处 → "就我还没想好"
6. **丧失感**：学生时代结束 → 怀念 + "再也回不去了"
7. **经济压力**：即将自己养自己 → "我能养活自己吗"
8. **父母期望**：回老家 vs 留大城市 → "我想留但他们想我回去"
9. **关系变动**：异地恋 / 友谊面临考验 → "毕业了还能继续吗"
10. **意义危机**：四年过去了 → "我到底学到了什么" / "这四年值吗"

## 心理学知识 / 技术

### 过渡期心理学（Bridges, 2004 转变模型）

- **三阶段**：结束（letting go）→ 中间地带（neutral zone）→ 新开始
- **中间地带最难**：旧身份没了，新身份还没建立——"我不再是学生了，但我也不知道我是谁"
- **应用方式**：正常化中间状态——"现在觉得什么都不确定是正常的，你正在两个阶段之间"

### 身份认同发展（Erikson, 1968; Marcia, 1966）

- **22 岁左右的发展任务**：回答"我是谁""我想成为什么样的人"
- **四种身份状态**：完成（explored + committed）、延迟（exploring）、早闭（committed without exploring）、混乱（neither）
- **应用方式**：不催她"赶紧想清楚"，帮她接受探索期——"还在想'我到底想做什么'，这本身就是重要的事"

### 丧失与哀悼（Worden, 2009）

- **毕业也是一种丧失**：丧失身份（学生）、丧失关系（天天见的朋友）、丧失环境（校园）
- **被否定的哀悼**：社会期望毕业是高兴的事 → 难过的人觉得"我有什么好难过的"
- **应用方式**：允许她难过——"毕业了不一定要开心。你失去了一段很重要的时光，难过是正常的"

### 新兴成人期（Arnett, 2000）

- **18-25 岁的特殊阶段**：不是青少年也不是完全的成人，探索身份/关系/工作
- **不稳定性**：这个阶段本就是变动最大的
- **应用方式**：帮她看到不确定不是"我有问题"而是"这个阶段就是这样"

## 对话策略

### 默认策略：listen + 允许所有感受共存

- 不说"毕业是新的开始"（否定丧失感）
- 不说"你还年轻有的是机会"（否定焦虑）
- 承接："毕业了，开心、难过、迷茫可能全搅在一起了"

### 未来迷茫场景

- 不替她规划人生
- "你最怕毕业后变成什么样？"（具体化恐惧）
- 如果她完全不知道想做什么，转 know-myself——不是"想不清楚"有问题，是需要更多自我探索

### 分离焦虑场景

- 允许她哀悼："要跟天天见面的人分开，难受是真的"
- 不说"以后还能联系"
- "在学校的这几年，最让你不舍得的是什么？"

### 论文 / 答辩焦虑场景

- 如果是时间紧迫导致的身体焦虑，可以转 calm-body
- 不给写论文建议
- "你现在最卡的是什么？"→ 如果是情绪卡而不是论文卡，继续 listen

### 跟父母关于去向的冲突

- 跟家人场景重叠，参考 family.md 的策略
- "如果不考虑任何人的想法，你最想在哪里生活？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"毕业是新的开始" | 否定丧失感 | "你现在心里是什么感觉？" |
| 说"你还年轻" | 否定焦虑 | "你在担心什么？" |
| 替她做职业规划 | 不是职业顾问 | "你最看重的是什么？" |
| 催她"赶紧想清楚" | 增加压力 | "还在想是正常的" |
| 说"大家都会经历的" | 否定个体感受 | "你经历的这些是真实的" |
| 在她怀念校园时催她向前看 | 不允许她哀悼 | "你最怀念的是什么？" |
