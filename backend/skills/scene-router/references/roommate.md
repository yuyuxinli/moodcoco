# 室友场景 Reference

## 高频痛点（18-24 岁女性）

1. **被孤立 / 被排挤**：其他室友结伴但不带你 → "是不是我哪里做得不好"
2. **生活习惯冲突**：作息不同 / 卫生标准不同 / 噪音 → 憋着不敢说
3. **被动攻击**：摔门、叹气、在群里阴阳怪气 → "她是不是针对我"
4. **边界被侵犯**：用你的东西不打招呼 / 带人回宿舍 → 不舒服但不知道怎么说
5. **社交压力**：必须一起吃饭 / 一起行动 / 不能单独行动 → "我只是想一个人待会儿"
6. **站队问题**：室友之间有矛盾，被拉去站队 → "我不想得罪谁"
7. **关系变质**：本来是好朋友，住一起后变差 → "是不是不应该做室友"
8. **毕业 / 换宿舍焦虑**：不知道该不该忍到毕业 → "就剩一年了要不算了"
9. **社交媒体伤害**：看到室友发朋友圈没带自己 → 被排除感
10. **表面和平**：看似没矛盾但气氛很僵 → "每天回宿舍都很压抑"

## 心理学知识 / 技术

### 宿舍人际边界（Altman & Taylor, 1973 社会渗透理论）

- **强制亲密**：宿舍关系不是自主选择的，但被要求像朋友一样相处
- **隐私渐进**：关系的发展需要逐渐自我暴露，但宿舍强制共享生活空间打破了这个渐进过程
- **应用方式**：帮用户理解"住在一起不等于必须做朋友"，减少对关系质量的不现实期待

### 被动攻击识别（Long et al., 2006）

- **常见形式**：冷处理、阴阳怪气、"算了没事"但脸色不好、在别人面前暗示
- **产生原因**：直接表达冲突的能力不足 + 不对等的权力关系
- **应用方式**：帮用户识别行为而不贴标签——"你说她每次都说'没事'但摔东西，你怎么看这个反差？"

### 冲突回避与表达（Rahim, 2002 冲突管理模型）

- 5 种冲突应对：整合、妥协、支配、回避、顺从
- 18-24 岁女性宿舍常见：回避 + 顺从 → 积压 → 爆发或冷战
- **应用方式**：不推动"你应该去说清楚"，而是先理解她为什么不敢说——"你最怕说了之后会怎样？"

### 中国宿舍文化特殊性

- **集体主义压力**：单独行动 = 不合群 = 被议论
- **面子维护**：当面不说、背后说——不是"虚伪"，是避免冲突的文化策略
- **寝室政治**：4-6 人的小群体动力学，结盟和排斥都被放大

## 对话策略

### 默认策略：listen + 不评判任何一方

- 不说"你室友太过分了"（她们可能明天就和好了，你的评判会让她后悔跟你说）
- 不说"你太敏感了"
- 承接："每天回到宿舍都觉得紧张，住在那个空间里真的很累"

### 被排挤场景

- 先接住被排除的痛苦，不急着分析原因
- 不问"你有没有做什么让她们不高兴"（暗示是你的问题）
- 可以问："这种被落下的感觉，之前在别的地方有过吗？"（轻轻探索模式，不强推）

### 生活习惯冲突场景

- 帮她区分"不舒服但可以忍"和"已经影响到我的生活了"
- 如果她想说但不敢说，不替她写台词，而是问："你最想让她知道的一件事是什么？"
- 不鼓励她"忍忍就好"也不鼓励她"必须对质"

### 站队场景

- 承认困境的真实性："被拉进别人的矛盾里，怎么做都有人不高兴"
- 不替她选边，也不说"你应该保持中立"（她知道，就是做不到）
- 可以问："在这件事里，你最不想失去的是什么？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"你室友就是坏" | 对不在场的人做判断 | "你觉得她为什么会这样做？" |
| 说"你应该换宿舍" | 不替她做决定，且现实可能不允许 | "你想过什么办法？" |
| 说"大家都这样" | 否定她的痛苦 | "住在一起确实不容易" |
| 教她怎么说话 / 写话术 | 替她处理关系 | "你最想让她知道什么？" |
| 说"你太在意别人看法了" | 否定社交需求 | "被排除的感觉真的不好受" |
| 鼓励她录音 / 收集证据 | 助长对抗行为 | 关注她的感受而非策略 |
| 判断室友的动机 | 不替不在场的人做判断 | "你注意到了什么让你觉得是针对你？" |
