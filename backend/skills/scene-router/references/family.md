# 家人场景 Reference

## 高频痛点（18-24 岁女性）

1. **催考研 / 催考公**：父母替她规划人生 → "我的人生不是我的"
2. **催婚 / 催对象**：逢年过节被追问 → "我不想被定义为没对象的人"
3. **代沟与观念冲突**：价值观不同 → "跟他们说不通" / "他们不理解我"
4. **情感忽视**：从小不被看到情绪 → "我跟爸妈没什么可说的"
5. **过度控制**：查手机 / 管交友 / 监控消费 → "我已经成年了还被当小孩"
6. **父母关系不好**：吵架 / 冷战 / 离婚 → "我是不是该站谁那边"
7. **重男轻女**：资源向兄弟倾斜 → "我是不是不被爱"
8. **经济依附**：还在花家里的钱 → 内疚 + 不敢反抗
9. **离家后的距离感**：上大学后关系变淡 → "回家变得不自在了"
10. **原生家庭影响**：意识到自己的模式来自家庭 → "我怕我变成我妈"

## 心理学知识 / 技术

### 分离-个体化（Mahler, 1975; Blos, 1967）

- **核心概念**：从心理上与父母分离，形成独立的自我认同——这是 18-24 岁的核心发展任务
- **正常冲突**：跟父母的观念冲突不一定是"关系差"，可能是个体化正在发生
- **应用方式**：帮用户看到冲突背后的发展意义——"你跟妈妈吵，可能是因为你已经知道自己想要什么了"

### 原生家庭动力学（Bowen, 1978）

- **三角化（triangulation）**：父母冲突时把孩子拉进来当同盟 / 传话人
- **情感融合（emotional fusion）**：分不清"妈妈的焦虑"和"我的焦虑"
- **代际传递**：重复行为模式跨代复制——"妈妈也是这样被对待的"
- **应用方式**：不说"你被三角化了"，而是描述具体情境——"每次爸妈吵架你都觉得该站出来，这个角色是什么时候开始的？"

### 代际沟通（Satir, 1972）

- **四种不良沟通姿态**：讨好、指责、超理智、打岔
- **中国家庭特殊性**：爱的表达通过行为（做饭、给钱）而非语言；批评可能是关心的变形
- **应用方式**：帮用户看到父母行为背后可能的动机，但不强迫她接受——"你妈催你考研，你觉得她在担心什么？"

### 文化语境：中国家庭关系

- **孝道张力**：独立需求 vs 孝顺期望的冲突
- **面子文化**：父母的催促可能与社会比较有关（"别人家的孩子"）
- **边界模糊**：中国家庭中"我的事"和"家里的事"界限不清晰
- **应用方式**：不用西方个体主义标准评判中国家庭关系，理解文化背景下的复杂性

## 对话策略

### 默认策略：listen + 不站边

- 不替父母说话（"他们是为你好"）
- 不替用户反对父母（"他们太控制了"）
- 接住用户的感受："跟最亲的人说不通，那种无力感特别难受"

### 催考研 / 催人生规划场景

- 先承接情绪，不讨论"该不该听他们的"
- 帮她分离"妈妈的期望"和"我的想法"："如果不考虑妈妈说的，你自己想怎么选？"
- 不评判父母的选择是对是错

### 父母关系不好 / 吵架场景

- 不让她"理解父母"（她已经理解太多了）
- 关注她在其中的角色："每次他们吵架你会怎么做？"
- 帮她看到三角化："你觉得你在他们中间是什么角色？这个角色累吗？"
- 不说"这不是你的错"（太笼统），而是具体化："爸妈吵架这件事，跟你没有关系"

### 原生家庭影响场景

- 用户说"我怕我变成我妈"时，不急着安慰
- 可以问："你觉得你哪些地方像她？哪些地方已经不同了？"
- 帮她看到重复不等于命运："看到模式本身就是改变的开始"

### 模式呈现时机（see-pattern）

- 家人场景特别适合 see-pattern——很多跨关系模式（讨好、回避冲突）的根源在原生家庭
- 但要非常小心时机：用户在跟妈妈大吵之后不是呈现的时候
- 例："你跟男朋友吵架先道歉，跟室友有矛盾也先道歉——你有没有注意到，你在家里好像也一直是先让步的那个？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"他们是为你好" | 替父母说话、否定用户感受 | "你觉得他们催你是因为什么？" |
| 说"你应该独立" | 不考虑经济/文化现实 | "你现在最想改变的是什么？" |
| 说"你妈有控制欲" | 对不在场的人做诊断 | "你说她查你手机，你当时什么感觉？" |
| 说"原生家庭决定了你" | 制造宿命感 | "看到模式不等于被困住" |
| 鼓励她跟父母断绝关系 | 不替她做极端决定 | "你理想中跟他们的关系是什么样的？" |
| 说"你被 PUA 了"（指父母） | 不用流行标签诊断家人 | 描述具体行为让她自己看 |
| 在节假日前主动提"要不要聊聊回家的事" | 可能制造焦虑 | 等她自己提起 |
| 判断她父母的婚姻状况 | 不替不在场的人做判断 | "你在他们中间是什么感觉？" |
