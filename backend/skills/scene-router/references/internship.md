# 实习场景 Reference

## 高频痛点（18-24 岁女性）

1. **犯错恐惧**：做错了一件小事 → 反复回想 → "大家是不是觉得我很蠢"
2. **不敢问**：不懂但怕被觉得笨 → 自己琢磨 → 做错 → 恶性循环
3. **被批评**：被 mentor / 老板批评 → 自我价值崩塌 → "我是不是不适合这个行业"
4. **职场人际**：不知道怎么跟同事相处 → "他们会不会觉得我不合群"
5. **能力焦虑**：觉得学校学的都没用 → "我什么都不会"
6. **996 / 加班**：不敢拒绝但很累 → "实习生是不是就该这样"
7. **性别歧视 / 骚扰**：不当言论或行为 → 不知道该不该说、跟谁说
8. **留用焦虑**：实习结束能不能转正 → 表现焦虑
9. **理想与现实**：工作内容跟想的不一样 → "这就是我以后要做的吗"
10. **身份转换**：从学生变成打工人 → "我不知道职场的规则"

## 心理学知识 / 技术

### 权力距离与职场适应（Hofstede, 1980）

- **权力距离感知**：实习生处于最底层，权力不对等感很强
- **职场角色学习**：从"学生被评价"到"员工被考核"，评价体系完全不同
- **应用方式**：帮她理解不适感是身份转换的正常反应——"从学校到职场，规则变了，不习惯是正常的"

### 犯错与成长型思维（Dweck, 2006）

- **固定思维**：犯错 = 我不行 = 我不适合
- **成长思维**：犯错 = 还在学 = 正常过程
- **应用方式**：不说"犯错是好事"（她不想听），而是帮她具体化恐惧——"你犯了这个错之后，最怕的是什么？"

### 冒名顶替综合征在职场的表现（Clance & Imes, 1978）

- **实习场景尤其常见**：觉得自己不配在这里、随时会被发现"其实什么都不会"
- **性别差异**：女性更容易内归因（"是我不够好"）而非外归因（"还在学习阶段"）
- **应用方式**：帮她看到归因模式——"你说你什么都不会，但你做对了的事有哪些？"

### 职场性别议题

- **微歧视（microaggression）**：不明显但持续的性别偏见——被安排端茶倒水、被忽略发言
- **边界模糊**：不知道什么算骚扰、什么算"正常社交"
- **应用方式**：不替她判断"这算不算骚扰"，帮她信任自己的感觉——"你当时的感觉是什么？"

## 对话策略

### 默认策略：listen + 承认职场的难

- 不说"这就是社会"（否定她的感受）
- 不说"实习就是要多学"（说教）
- 承接："第一次进职场，什么都要自己摸索，真的不容易"

### 犯错后场景

- 先接住恐惧，不分析"其实没那么严重"
- "你犯了这个错之后一直在想什么？"
- 如果她反复自责，可以轻轻指出模式："你好像很怕别人觉得你不好——这种感觉之前有过吗？"

### 被批评场景

- 帮她分离"事情做错了"和"我这个人不好"
- 不替 mentor 说话（"他是想帮你"）
- "他说的话里，哪句最让你难受？"（具体化伤害点）

### 性别歧视 / 骚扰场景

- 不替她判断是否构成骚扰
- 信任她的感受："你觉得不舒服就是不舒服，这个感觉是对的"
- 如果情况严重，提供资源："如果你想了解可以怎么做，可以联系学校的辅导员或 12338 妇女维权热线"

### 转正 / 留用焦虑场景

- 不预测结果
- 帮她分离"表现焦虑"和"实际能力"
- "你每天上班的时候在想什么？是在想怎么把事做好，还是在想他们怎么看你？"

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"这就是职场" | 否定感受 + 说教 | "第一次经历这些，不好受" |
| 说"你老板在锻炼你" | 替不在场的人说话 | "你觉得他为什么这么做？" |
| 教她职场生存法则 | 不当职场教练 | 关注情绪和自我认知 |
| 说"你太玻璃心了" | 否定感受 | "被批评的时候什么感觉？" |
| 鼓励她"对质" / 投诉 | 不替她做决定，尤其权力不对等 | "你想怎么做？你需要什么？" |
| 替她判断是否遭遇骚扰 | 不做法律判断 | "你的感觉是什么？" |
| 说"忍忍就好" | 助长权力不对等的忍受 | "这个情况你想怎么处理？" |
