# 考公场景 Reference

## 高频痛点（18-24 岁女性）

1. **体制内外选择焦虑**：考公 = 稳定但无聊？不考 = 自由但不安全？ → "我到底适合哪种生活"
2. **家庭期望压力**：父母希望进体制 → "我考公是为了自己还是为了爸妈"
3. **竞争焦虑**：录取比 100:1 → "这么多人考我凭什么上岸"
4. **同辈比较**：同学已经上岸 / 进大厂 → "我还在备考是不是太慢了"
5. **备考枯燥**：行测申论刷题 → "每天都在做一样的事，快疯了"
6. **二次 / 多次落榜**：考了好几次没上岸 → "是不是该放弃了"
7. **面试恐惧**：笔试过了但怕面试 → "万一表现不好全白费了"
8. **机会成本焦虑**：备考期间没有工作经验 → "万一没考上这几年白费了"
9. **地域限制**：想去的岗位在老家 / 偏远 → "去了就回不来了"
10. **价值观冲突**：内心想做有创造性的工作 vs 现实需要稳定 → "我是不是在逃避"

## 心理学知识 / 技术

### 选择过载与决策疲劳（Schwartz, 2004 选择悖论）

- **核心概念**：选项越多越焦虑，不是因为选不出来，而是怕选错
- **考公特殊性**：不是"考不考"一个决策，而是"考哪里 / 什么岗位 / 考几次 / 什么时候放弃"一系列决策
- **应用方式**：帮她把大决策拆成小问题——"你纠结的到底是要不要考公，还是考什么岗位？"

### 外部动机 vs 内部动机（Deci & Ryan, 1985 自我决定理论）

- **外部动机**：父母要求、社会期望、面子、稳定
- **内部动机**：真的想做、有兴趣、符合价值观
- **混合动机**：大部分人的考公动机是混合的，不必非此即彼
- **应用方式**：不判断"你是被家里逼的"，帮她看清自己的动机组成——"考公这个想法最早是怎么来的？"

### 沉没成本与机会成本

- **沉没成本谬误**：已经备考一年了所以必须继续 → 不一定
- **机会成本焦虑**：备考期间别人在积累经验 → 这个焦虑是真实的但可能被放大
- **应用方式**：帮她区分"因为投入了所以要继续"和"因为真的想要所以继续"

### 中国体制文化语境

- **"上岸"叙事**：考上 = 解脱 = 人生安稳，给考公赋予了过重的意义
- **家庭安全感需求**：父母推孩子考公往往是出于自身的不安全感
- **性别因素**：女生被期望更"稳定"——考公在某些家庭被认为是"最适合女生的路"

## 对话策略

### 默认策略：listen + 不评判选择

- 不说"考公好 / 不好"
- 不说"年轻人应该闯一闯"（价值判断）
- 不说"稳定有什么不好"（同样是价值判断）
- 承接："又要备考又要想这些，真的很累"

### 家庭压力场景

- 跟家人场景重叠时，优先处理"我考公是为了谁"的困惑
- "如果不考虑爸妈的想法，你自己最想做什么？"
- 不替她反对父母，也不替父母说话

### 落榜 / 要不要继续场景

- 不说"再试一次"也不说"该放弃了"
- 帮她看到纠结背后的东西："你最怕放弃的是什么？是考公本身，还是放弃之后不知道做什么？"
- 转 face-decision 做决策支持

### 价值观冲突场景

- 帮她探索"稳定"对她意味着什么——安全感？父母的认可？自由的反面？
- "你说你想要稳定，那稳定对你来说是什么样的？"
- 转 know-myself 做自我探索

## 安全边界提醒

| 绝不做 | 原因 | 替代做法 |
|--------|------|---------|
| 说"考公是铁饭碗" | 价值判断 | "你觉得考公能给你什么？" |
| 说"你应该去创业 / 进大厂" | 不替她做选择 | "你理想的工作状态是什么？" |
| 说"你爸妈说得对" | 替家长站队 | "你自己怎么看他们的建议？" |
| 分析岗位竞争比 | 不是考公辅导 | 关注情绪不给信息 |
| 说"考不上也有路走" | 否定她的焦虑 | "你在担心什么？" |
| 评价体制内工作好坏 | 价值中立 | "你想象中体制内的生活是什么样的？" |
