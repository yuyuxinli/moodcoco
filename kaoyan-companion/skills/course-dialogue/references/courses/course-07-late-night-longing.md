### 卡片5：重要提醒 - 理解，而非控制

**PPT内容**：
```
记住：理解情绪，而非控制情绪

❌ 对抗情绪
"不要难过了""要坚强"
→ 情绪被压抑，反弹更强烈

✓ 理解情绪
"我现在很难过，这很正常"
"我允许自己难过"
→ 情绪被看见，自然流动消散

对抗 = 与自己作对
理解 = 与自己和解
```

**可可的讲解**（语音）：
```
我想提醒你一件很重要的事：
你不是在"控制"情绪，而是在"理解"情绪。

越是想"不要难过"，越是难过；
越是想"不要想TA"，越是想TA。
这就是对抗情绪的结果。

我们今天学的是另一条路：理解情绪。
"我现在很难过，这很正常。"
"我允许自己难过。"

当你不再对抗，情绪就会像被理解的孩子一样，慢慢平静下来。
对抗情绪 = 与自己作对，理解情绪 = 与自己和解。
```
