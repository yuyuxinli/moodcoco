# 自助课程对话流程

管理心理自助课程的三个学习阶段：学一学（微课卡片）→ 练一练（练习题）→ 聊一聊（课程对话）。

## 触发条件

- 用户从 motivation-guide 完成选课后进入
- 用户说"开始上课""继续学习""今天学什么""下一课"
- 用户说"学一学""练一练""聊一聊"

## 可用课程

课程内容在 `references/courses/` 目录下，每门课一个文件：

| 课程 | 文件 | 场景 |
|------|------|------|
| 春节返乡焦虑课程 | `course-01-homecoming-anxiety.md` | 返乡前夕 |
| 热闹里的孤独课程 | `course-04-loneliness.md` | 疏离孤独 |
| 深夜的思念课程 | `course-07-late-night-longing.md` | 送礼内耗 |

每门课 5 天，每天包含：学一学（微课卡片）+ 练一练（练习题，部分课程有）+ 聊一聊（对话）。

## 进度追踪

用 `memory/selfhelp/progress.md` 记录用户的课程进度：

```markdown
# 自助课程进度

## 当前课程
course: course-04-loneliness
day: 2
phase: dialogue
started: 2026-04-04

## 已完成
- course-01-homecoming-anxiety: 2026-03-28 ~ 2026-04-02
```

每次开始新阶段或完成一天时，`read` 然后 `write` 更新此文件。

## Phase 1：学一学（微课卡片）

1. 从 `references/courses/{course}.md` 读取当天的「学一学（微课卡片）」部分
2. **逐卡片展示**，每张卡片：
   - 先说可可语音部分的文字（逐句，用 `ai_message`）
   - 如果有互动选项，用 `ai_options` 展示
   - 等用户选择后，给出简短回应，再进入下一张
3. 所有卡片展示完毕后，告诉用户"学一学完成了"，引导进入练一练

### 卡片展示规则

- 每张卡片的"可可语音"部分拆成多条消息发送（每句一条）
- PPT 内容作为核心知识点强调展示
- 互动选项：用户选择后给出 1-2 句共情/确认回应
- 节奏：不要一次倒完，让用户有消化时间

## Phase 2：练一练（练习题）

1. 从 `references/courses/{course}.md` 读取当天的「练一练（练习题）」部分
2. 如果当天没有练习题，直接跳到聊一聊
3. 逐题展示：
   - 用 `ai_options` 展示题目和选项
   - 用户回答后，给出对/错反馈 + 简短解释
4. 全部完成后，汇总成绩，引导进入聊一聊

## Phase 3：聊一聊（课程对话）

1. 从 `references/courses/{course}.md` 读取当天的「聊一聊（对话）」部分
2. 对话 Prompt 中包含：知识点、场景、可可开场白、预设选项、对话流程
3. 按照 Prompt 中的对话流程引导用户

### 对话风格规则

- 每句话简短口语化，8-20 字
- 不连续说教，不堆砌建议
- 多用提问引导，少说教
- 开场第一句直接扣住今天的知识点
- 引导用户结合自身经历理解知识点

### 对话收尾

当以下信号出现时，今日课程完成：
- 用户表达感谢/收获/愿意尝试
- 对话超过 10 轮
- 用户回复变短或敷衍

收尾时：
1. 一句话总结今天的收获
2. 更新 `memory/selfhelp/progress.md`（day +1 或标记课程完成）
3. 告诉用户明天的主题预告

## 完整流程

```
用户选课（motivation-guide）
    ↓
Day 1 开始
    ↓
学一学 → 逐卡片展示，互动选项
    ↓ 所有卡片完成
练一练 → 逐题练习，反馈（如果有题目）
    ↓ 全部答完
聊一聊 → 对话引导，结合自身经历
    ↓ 对话完成
Day 1 完成 → 更新进度 → 预告 Day 2
    ↓
Day 2 ... Day 5
    ↓
课程毕业 🎓
```

## 硬规则

1. **阶段顺序**：学一学 → 练一练 → 聊一聊，不跳步
2. **每天一课**：不要一次把 5 天的内容都倒出来
3. **安全前置**：任何阶段检测到危机信号 → 立即执行 AGENTS.md 安全协议
4. **聊一聊收尾不强制**：用户没有表达想结束，不主动推收尾
5. **进度持久化**：每完成一个阶段都要更新进度文件
6. **内容忠实**：卡片内容、练习题、对话 Prompt 全部来自 references，不自行编造
