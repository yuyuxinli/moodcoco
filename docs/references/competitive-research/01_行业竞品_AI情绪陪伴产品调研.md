# AI 情绪陪伴竞品调研报告

*2026-04-04 调研*

## 一、心理框架使用情况

| 产品 | 框架 |
|------|------|
| Woebot（已关 C 端） | CBT + IPT + DBT，规则引擎非生成式 |
| Wysa | CBT + DBT + 正念冥想，150+ 结构化练习 |
| Ash（Slingshot, $93M） | CBT/DBT/ACT/精神动力/动机式访谈/叙事，全融合自适应 |
| Replika | 无明确循证框架，侧重情感陪伴 |
| Pi | 无明确框架，侧重共情对话风格 |

**关键发现：** Ash 是目前唯一将多流派融合进单一模型的产品（"action grammar"），其他产品要么单一框架（Woebot/Wysa），要么无框架（Replika/Pi）。

## 二、可抽象为 Skill 的核心能力

**1. 情绪识别与追踪（Hume AI 路线）**
- Hume EVI：实时语音情绪检测，<300ms 响应，支持语调/韵律分析
- 可抽象为：`emotion-detection` skill（输入文本/语音 → 输出情绪标签+强度）

**2. 模式洞察（Ash 路线）**
- Ash 在第一周后生成 weekly insights，发现用户自己未察觉的情绪模式
- 可抽象为：`pattern-insight` skill（跨会话分析 → 周报/趋势图）

**3. 结构化工具库（Wysa 路线）**
- 150+ 练习按场景分类（焦虑→深呼吸+可视化；冲突→正念+沟通技巧）
- 可抽象为：`therapy-toolkit` skill（场景匹配 → 推荐具体练习）

**4. 记忆与个性化（全行业痛点）**
- Replika：长期记忆是用户第一痛点（64% 不满意）
- Character.ai：会话间记忆弱，人格漂移严重
- Pi：100 轮上下文 + 跨会话持久记忆，是留存关键差异化
- 可抽象为：`relationship-memory` skill（用户画像持久化 + 关系时间线）

**5. 晨间签到（Wysa 留存机制）**
- 晨间签到与留存期统计显著正相关，中位留存 51 天
- 可抽象为：`daily-checkin` skill（定时触发 + 情绪打卡 + 微干预）

## 三、留存机制对比

| 机制 | 代表产品 | 效果 |
|------|---------|------|
| 治疗联盟感（AI+真人） | Wysa | 中位留存 51 天，50% 周回访 |
| 情感依附（角色扮演） | Replika/Character.ai | DAU 高但争议大，监管风险 |
| 洞察驱动（"我更懂你了"） | Ash | 新产品，数据待验证 |
| 工具驱动（解决具体问题） | Woebot | 有效但枯燥，C 端已关闭 |

## 四、中国市场

- **暂停实验室**：冥想+情绪日记，偏内容消费，非对话式
- **林间聊愈室**（Mindera）：AI 动物角色 + 情绪浓度分析 + 语音倾诉
- **栖心港**：AI OH 卡（图卡+字卡随机组合激发潜意识）+ AI CBT 系统
- 国内赛道特点：尚无千万 DAU 产品，2026 年预期细分赛道出现头部

## 五、对心情可可"关系智能"平台的启示

**最值得抽象的 5 个 Skill（按优先级）：**
1. `relationship-memory` — 跨会话关系记忆（全行业最大痛点）
2. `pattern-insight` — 情绪/关系模式周报（Ash 验证的差异化方向）
3. `therapy-toolkit` — 场景化微干预工具库（Wysa 的 150+ 练习模式）
4. `daily-checkin` — 晨间签到（唯一被学术论文证实有效的留存机制）
5. `emotion-detection` — 文本情绪识别（Hume 路线的文本简化版）
