# 可可下一阶段产品方向（周二会议准备）

*2026-03-07 整理*

---

## 核心框架：留存 = 有用 + 有趣 + 有牵挂

不追求 Tolan 式的游戏世界系统，而是让对话本身变得有用、有趣、有牵挂。视觉/听觉/操作性作为对话的增强层——在对的时刻、用对的形式，放大效果。

---

## 三维 Benchmark（每个维度 5 级，LLM 可评估）

### 有用（已有 benchmark）

情绪识别 → 共情回应 → 认知重构 → 洞察引导 → 行动支持

### 有牵挂

| 维度 | 评什么 | 1 分 | 5 分 |
|------|--------|------|------|
| ① 人格辨识度 | 可可是否有独特的「人」的感觉 | 通用 AI 味，换成豆包毫无违和 | 用户能从 10 段回复中一眼认出可可 |
| ② 记忆编织 | 是否自然地用了用户历史 | 完全不引用 / 机械复述 | 像老朋友一样自然关联过去的经历 |
| ③ 关系阶段感 | 亲密度是否匹配关系深度 | 第 50 次对话还像第 1 次一样客气 | 有专属梗、简称，语言随关系递进 |
| ④ 主动牵挂 | 可可是否关心用户没提的事 | 永远只回应用户主动提到的 | 「上次你说要面试，怎么样了？」 |
| ⑤ 自我暴露 | 可可是否有自己的内心世界 | 纯镜像，只反映用户情绪 | 分享自己的感受、想法、发现 |

递进逻辑：有人格 → 记得你 → 关系在长 → 惦记你 → 她也对你敞开

理论基础：Parasocial Relationship Theory + Social Penetration Theory + Attachment Theory

### 有趣

| 维度 | 评什么 | 1 分 | 5 分 |
|------|--------|------|------|
| ① 表达感染力 | 语言是否生动有画面感 | 「我理解你的感受，这很正常」 | 「那种感觉就像你明明握着手机，但还是到处找手机」 |
| ② 意外性 | 回复是否有「没想到」的元素 | 用户说难过→「抱抱你」100% 可预测 | 出其不意但恰到好处，换角度、提问、类比 |
| ③ 互动模式多样性 | 是否超越「你说我听」 | 永远倾听→共情→建议 | 提问、小实验、角色扮演、故事、「如果...」 |
| ④ 好奇心制造 | 是否让用户想继续聊 | 回复是终结句 | 留钩子、伏笔、让用户想验证的想法 |
| ⑤ 探索空间 | 用户是否可以带对话去任何方向 | 话题锁死在情绪处理上 | 哲学、八卦、梦、奇怪想法，可可都能接住 |

递进逻辑：说话有意思 → 不可预测 → 玩法多 → 聊完还想聊 → 什么都能聊

理论基础：HAP Model (Variety + Appreciation) + SDT (Autonomy + Competence) + Information Gap Theory

## 视觉不是世界，是对话的增强层

核心原则：**每个组件的存在理由是放大那个对话瞬间的情感强度。**

不做 Tolan 式的持久世界系统。视觉组件是临时的、上下文相关的，在关键时刻出现，强化效果后消失。天然适配微信小程序的富媒体消息流。

### 有用的组件
- 情绪识别后弹一张「情绪卡片」——有画面感的图 + 一句话命名情绪
- 认知重构时，可滑动的「视角切换卡」——左滑当前想法，右滑另一个角度
- 行动建议变成「今日处方」卡片，可保存到相册

### 有趣的组件
- 可可发「手写便签」风格图片，写一个跟对话相关的奇怪问题
- 聊到某个比喻时，生成即时插画——对话内容的视觉化
- 翻牌互动——可可说「我猜你现在...」翻开看对不对

### 有牵挂的组件
- 隔几天可可发一张「给你的信」——信纸样式，基于记忆写
- 关系里程碑卡片：「这是我们第 30 次聊天」+ 可可写的一段感受
- 可可的「今日心情」——主动分享一个小状态

## 操作性：不做游戏控制，做对话手势

问题不是缺操作自由度，是**用户看不到自己说话的后果**。

解法：让对话的后果可见——用户说话 → 可可有反应 + 环境有变化（即时可见）。

### 两个核心手势

**双击可可** = 「你说点什么吧」
- 用户打开小程序不知道说什么 → 双击可可 → 她主动开口
- 可可说什么基于上下文：时间、近期对话、记忆、她的「心情」
- 零认知负担，解决「想来但没话题」的真实场景

**左右滑动** = 「换一个话题」
- 不感兴趣 → 左滑 → 可可换一个话题
- 感兴趣 → 点进去 / 直接打字接话
- 类似刷短视频的操作，零学习成本（用户已被训练过）

### 探索循环

```
双击可可 → 她开口说一个话题
不感兴趣 → 左滑 → 换一个
感兴趣  → 停下来 → 进入对话
```

### 可可主动抛出的卡片类型
- 一个问题：「你有没有想过，为什么每次吵完架你都先道歉？」
- 一个记忆回调：「上次你说想学吉他，后来去了吗？」
- 一个可可的心情：「我今天看到一句话，觉得特别像你」
- 一个小工具：「要不要试试 3 分钟呼吸？」
- 一个洞察：「最近三次聊天你都提到了'不够好'，你注意到了吗？」

每张卡片挂在三个维度上，滑动数据成为无声的偏好信号。

## 反馈可见性（解决「对着墙说话」的感觉）

### 对话中即时反馈
- 用户表达强烈情绪 → 可可表情/动作变化
- 聊到某个深度 → 背景色/氛围微妙变化
- 用户说了有洞察的话 → 可可「停顿」→「等一下，你刚才说的那句话...」

### 对话后累积反馈
- 每次聊完，情绪图谱多一个点（视觉慢慢生长）
- 可可的记忆以可浏览形式存在——用户能翻，看到关系的痕迹
- 关系里程碑自然出现——不是触发的，是聊出来的

## 与 Tolan 的差异化

不在娱乐性上和 Tolan 竞争，而是让心理学深度变得有趣（类比 Inside Out 用 Pixar 的娱乐让心理学可触及）。

| | Tolan | 可可 |
|---|---|---|
| 视觉方向 | 持久世界（星球/装饰/建造） | 对话增强层（临时组件/反馈变化） |
| 操作性 | 游戏式控制（布置/装修） | 对话手势（双击/滑动） |
| 自主感来源 | 向外定制（外星人世界） | 向内探索（自己的内心） |
| 用户感受 | 「我在打造一个世界」 | 「可可真的懂我，而且和她聊天很有意思」 |

## 6 个月路线建议

### M1-M3：有牵挂（让可可活起来）
- 人格重设计（维度①②）
- 主动消息 + 记忆运用（维度③④）
- 双击可可 + 左右滑动手势上线
- 联合创始人角色：设计可可的人格、说话方式、主动消息的内容调性

### M3-M6：有趣（让对话不无聊）
- 表达感染力 + 意外性提升（维度①②，prompt 工程）
- 互动组件逐个上线（有用/有趣/有牵挂各 2-3 个）
- 反馈可见性（表情变化、氛围变化）
- 联合创始人角色：设计组件的视觉/听觉表现、互动细节

### M6+：看数据决定
- 哪个维度分数最低就补哪个
- 如果「探索空间」成为瓶颈，再考虑更大的系统

## 理论支撑速查

| 理论 | 核心观点 | 对应维度 |
|------|----------|----------|
| HAP Model (Sheldon & Lyubomirsky 2012) | Variety + Appreciation 防止快感衰退 | 有趣：意外性、模式多样性 |
| SDT / PENS (Ryan & Deci 2000) | Autonomy + Competence + Relatedness → 内在动机 | 有趣：探索空间；有牵挂：关系感 |
| Parasocial Relationship Theory | 从工具使用 → 准社交 → 情感依恋的递进 | 有牵挂：全部 5 个维度 |
| Social Penetration Theory | 关系通过互相自我暴露逐层加深 | 有牵挂：关系阶段感、自我暴露 |
| Information Gap Theory (Loewenstein 1994) | 好奇心 = 已知与未知之间的缺口 | 有趣：好奇心制造 |
| Narrative Identity (McAdams 2001) | 自我叙事中的 agency 预测幸福感 | 有用：洞察引导、行动支持 |
