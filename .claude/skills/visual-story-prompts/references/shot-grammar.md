# Shot Grammar for Card Stories

Principles from "Understanding Comics" (Scott McCloud) and "Master Shots" (Christopher Kenworthy), adapted for multi-card image story production.

## From "Understanding Comics"

### Panel Transitions (Chapter 3)

McCloud identifies 6 types of panel-to-panel transitions. For card stories, 4 are relevant:

| Type | Definition | Card Story Application |
|------|-----------|----------------------|
| **Moment-to-moment** | Tiny time increment, same subject | Adjacent character cards with only expression/posture change. Same scene, same angle, minimal change = slow, emotional pacing. |
| **Action-to-action** | Same subject, progressive action | Character picks up phone → reads → puts it down. Scene stays, action progresses. |
| **Scene-to-scene** | Jump across space/time | Character card → Panel card (different visual world). Requires significant closure from reader. |
| **Aspect-to-aspect** | Different facets of same moment | Show the room, then show the character's hands, then show the window. Builds atmosphere. |

**Key insight**: Most card stories overuse scene-to-scene transitions (every card = new scene). Mix in moment-to-moment and aspect-to-aspect to create breathing room and emotional depth.

### Closure (Chapter 3)

Closure = the reader's imagination filling the gap between two images.

- **Small gap** (moment-to-moment): Reader barely notices the transition. Use for emotional sequences where immersion matters.
- **Medium gap** (action-to-action): Reader follows the logic. Use for narrative progression.
- **Large gap** (scene-to-scene): Reader must reconstruct context. Use sparingly — every scene-to-scene transition risks losing the reader.

**For cross-style transitions** (character → panel), the gap is inherently large because the visual world changes. Bridge rules compensate by reducing the closure burden through shared visual elements.

### Time-as-Space (Chapter 4)

In comics, the size and shape of a panel affects perceived time:

| Panel Property | Time Effect | Card Story Equivalent |
|---------------|-------------|----------------------|
| Tall narrow panel | Quick, tense moment | Character card with tight framing, little environment |
| Wide panel | Lingering, atmospheric | Character card with generous environment, breathing room |
| Borderless/bleeding | Timeless, infinite | Text-focused card with pure background, no frame |
| Small inset | Brief aside | Panel card's expression boxes |

**Application**: When a story moment should feel slow and heavy, use wide framing with environment. When it should feel sharp, use tight close-up with minimal background.

## From "Master Shots"

### Shot Size and Emotional Distance (Vol 1, Chapters 1-3)

| Shot Size | Frame Content | Emotional Effect | When to Use |
|-----------|--------------|-----------------|-------------|
| **Extreme wide** | Full environment, tiny figure | Isolation, overwhelm, context | Opening establishing shot, loneliness moments |
| **Wide/Full** | Full figure + environment | Observer distance, calm | Scene transitions, breathing room |
| **Medium** | Waist up + some environment | Natural conversation distance | Default narrative, most story cards |
| **Medium close** | Chest up | Engaged, attentive | Emotional dialogue, key story beats |
| **Close-up** | Face only | Intimacy, intensity | Emotional peaks, critical reactions |
| **Extreme close-up** | Single feature (eyes, hands) | Maximum intensity, symbolic | Turning points, detail emphasis |

**Key rule**: Shot size = emotional distance. Moving closer = increasing intimacy. Moving back = providing context or breathing room.

### Camera Angle and Power (Vol 1, Chapters 4-6)

| Angle | Effect | When to Use |
|-------|--------|-------------|
| **Eye-level** | Neutral, equal | Default for most cards |
| **Slight high angle** (looking down) | Vulnerability, smallness | Character feeling overwhelmed, uncertain |
| **Slight low angle** (looking up) | Strength, determination | Character making a decision, CTA moments |
| **Dutch angle** (tilted) | Unease, disorientation | Avoid in warm/healing stories |

### Directional Continuity (Vol 2, Chapter on dialogue)

When two shots cut between characters/scenes:

- **Match gaze direction**: If character A looks right, character B should look left (conversation feel)
- **Match action direction**: If movement flows left-to-right, maintain it across cuts
- **180-degree rule**: Stay on one side of the action axis to avoid disorienting the viewer

**Card story adaptation**: When a character card transitions to a panel card:
- Character's gaze should point toward where the panel card's main content will be
- Panel card's content layout should feel like "what the character is seeing"
- Return card should show character from the same spatial relationship (don't flip the room)

### Emotional Rhythm Through Shot Patterns (Vol 3)

Kenworthy identifies common shot patterns for emotional sequences:

**Build-up pattern**: Wide → Medium → Close → Close → Medium
- Use for: Approaching emotional climax

**Release pattern**: Close → Medium → Wide
- Use for: After emotional peak, providing breathing room

**Pulse pattern**: Medium → Close → Medium → Close → Medium
- Use for: Sustained tension or repeated emotional hits

**Card story application**: Map these patterns onto character-focused cards across the story. The pattern guides shot size assignment in Step 4 of the workflow.

## Combined Framework: The Continuity Stack

For card stories, visual continuity works in layers:

```
Layer 1 — COLOR CONTINUITY (always active)
  Color curve running through all cards. Never broken.

Layer 2 — SPATIAL CONTINUITY (within same-style runs)
  Same room, same furniture, same window. Camera moves around a fixed space.

Layer 3 — ELEMENT CONTINUITY (across style switches)
  Props, particles, light direction cross from one style to another.

Layer 4 — RHYTHMIC CONTINUITY (overall story)
  Shot sizes, style switching frequency follow emotional rhythm patterns.
```

When a transition feels jarring, diagnose which layer broke:
- Colors jumped? → Layer 1 fix (adjust background color)
- Lost sense of place? → Layer 2 fix (add spatial anchor)
- Styles feel disconnected? → Layer 3 fix (add crossing element)
- Pacing feels off? → Layer 4 fix (adjust shot sizes or reorder cards)
