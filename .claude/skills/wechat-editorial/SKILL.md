# 公众号正文编辑方法论

与 `social-content` 互补：social-content 管策略和结构，本 Skill 管逐字逐句的文本质量。

## 核心原则

**站在中国 Z 世代读者角度写每一个字。** 读者是 18-24 岁中国女性，没有留学背景，不关心学术出处，只关心"这跟我有什么关系"。

## 检查清单（逐段过）

### 1. 文化适配

- **外国文化必须翻译成中国人能懂的语境**。不是翻译语言，是翻译文化背景。
  - 错：Blue Monday → Brew Monday（英文双关，中国读者完全看不懂）
  - 对：把整个故事用中文逻辑重新讲一遍，去掉英文梗
- **外国场景必须桥接到中国场景**
  - 错："便利店阿姨跟你聊两句"（中国便利店没这文化）
  - 对："室友什么都没问，给你倒了杯热水"
- **外国词汇处理规则**：
  - 有通用中文译名的直接用中文：维生素D（不写 Vitamin D）
  - 没有译名的外文概念：先写中文含义，括号里放原文。如：`挪威人有个词叫 Friluftsliv，大意是"不管多冷都得出门走走"`
  - 人名保留原文可以，但全文不超过 3 个外国人名
- **不要假设读者知道外国常识**
  - 错："英国冬天日照很短"（读者不知道英国在什么纬度）
  - 对：要么解释为什么短，要么直接去掉这个信息点

### 2. 逻辑链完整性

**每一步推理都不能跳。** 读者不会自己补脑。

- 错："泡茶 → 所以有人陪了"（跳了）
- 对："泡茶需要几分钟，等水烧开的时候，人就坐下来了。坐下来了，就有人陪了。"
- 错："大家都很丧 → 卖机票"（为什么丧就要买机票？）
- 对："大家都这么丧，肯定想逃去暖和的地方度假。怎么让更多人买机票呢？"

**自查方法**：每两句话之间问"所以呢？"，如果答案不明显，中间缺了一步。

### 3. 章节间一致性

- **同一篇文章内不能自相矛盾**
  - 第三章说"见不到太阳不一定会丧" → 第四章就不能说"因为见不到太阳所以很丧"
- **概念/国家不能跨章节重复**
  - 如果第二章已经用日本举例说明"问题"，第三到七章的"解法"就不能再出现日本
  - 解法：明确分组。问题组（日本/韩国/俄罗斯）vs 解法组（挪威/英国/印度/芬兰/丹麦），零交集

### 4. 中文表达自然度

- **不要翻译腔**
  - 错："烧水壶"（这是"kettle"的直译，中文说"泡壶茶"或"烧壶水"）
  - 错："自由空气生活"（Friluftsliv 直译，绕口）
- **标点规范**
  - 禁用破折号（——），用逗号或句号断句
  - 引号用中文弯引号（""）
  - 引号只在标注引用或特殊含义时使用，不滥用
- **打破平行结构**
  - 错：四个连续的"有时候……有时候……有时候……有时候……"
  - 对：混合句式，长短交替，节奏变化
- **避免导航词**
  - 错："接下来，让我们看看英国的做法"
  - 对：直接开始讲英国

### 5. 场景沉浸 vs 分析

- **沉浸段要待够再走**。不要写两句画面就跳去讲道理。
  - 让读者先"进入"场景：感官细节（翻身、屏幕亮光、被子）、内心独白、具体时间
  - 至少 3-5 句纯场景描写，再过渡到分析
- **过渡要自然**，不要用"但你可能不知道的是"这种模板句
- **从个人到群体的过渡**：先让读者觉得"这说的是我"，再告诉她"不只是你"

### 6. 研究引用的克制

- **直觉能理解的道理不要引用研究**
  - 错："斯坦福大学的研究表明，跟陌生人聊天能缓解孤独感"
  - 对：直接说"有个人在旁边待着，够了"
- **只在反直觉时引用**：读者会"咦？"的时候才需要研究背书
  - 值得引用："零下20度出门散步反而能改善情绪"（反直觉，需要解释）
  - 不值得引用："有人陪着会好受一点"（常识）
- **引用时去掉学术味**
  - 错："2019年Leibowitz等人在《Journal of Affective Disorders》发表的研究表明"
  - 对："有个实验让一群人在零下的天气里出门走了20分钟"

### 7. 时间线与事实核查

- **涉及具体时间必须查证**
  - 春节放假几天、返工日期、补班安排——查国务院通知或已有素材
  - 不能凭感觉写"快两周了"，实际可能只有五天
- **外国节日/文化事件也要核实**
  - Blue Monday 的来源、年份、涉及的机构——查素材文档

## 审稿工作流

1. **逐段读**，不要一口气读完再改
2. 每段按上面 7 项清单过一遍
3. 发现问题先说出来，等确认后再改
4. 改完一段发完整段落给用户 review
5. 所有段落改完后，通读全文检查章节间一致性
