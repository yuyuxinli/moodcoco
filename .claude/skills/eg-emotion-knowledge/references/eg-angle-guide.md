# 可可 EG 视角内容创作指南

## 概述

本指南帮助基于 EG（情绪颗粒度）理论为心情可可创作内容。核心原则：**所有内容从情绪体验入手，不从概念定义入手。**

EG 不是一个独立的内容主题，而是心情可可所有内容的底层视角。无论写什么心理学话题，都用 EG 的方法"切第一刀"：先让读者感受到那个情绪，再帮她命名，最后给她行动。

---

## EG 内容创作三板斧

### 1. 从体验入手（Experience First）

**原则：** 先唤起读者的身体感受和场景记忆，再引入命名。

- 不要这样开头："今天我们来聊聊什么是焦虑"
- 应该这样开头："你有没有过这种感觉：明明什么都没发生，但心里就是不安..."

**为什么有效：** Barrett 的情绪构建论指出，情绪从身体感觉（interoception）开始。唤起身体感受 = 激活读者的情绪构建过程 = 她会觉得"这说的就是我"。

**操作要点：**
- 开头必须是一个具体场景或身体感受，不是定义或概念
- 用"你有没有过..."/"你知道那种...的感觉吗"等句式触发回忆
- 场景要足够具体：有时间、有地点、有身体感觉（胸口发紧、胃里发空、喉咙堵住）
- 避免上来就用情绪大词（焦虑、抑郁、内耗），这些词会让读者进入"认知模式"而非"体验模式"

**理论依据：** Barrett《How Emotions Are Made》— 情绪是大脑基于身体感觉和过往经验的主动构建

### 2. 提供精准词汇（Precise Labeling）

**目标：** 让用户获得"就是这个词！"的顿悟感（Aha Moment）。

**操作要点：**
- 从模糊大词引导到精准小词，制造认知升级感
- 框架句式："你以为你是[大词]，其实你是[精准词]"
- 给出精准词后，用一句话解释这个词的独特含义（和大词的区别在哪）
- 如果有合适的英文对应词，可以附上（增加"专业感"和"发现感"）

**为什么有效：** Brown 的命名即赋权（Name it to claim it）+ Barrett 的 EG 研究都证实：能精准命名一种感受的人，情绪调节能力显著更强。命名本身就是干预。

**精准词汇的来源：**
- `emotion-landscape.md`：87 种情绪体验，按 Mood Meter 四象限组织
- `emotion-differentiation.md`：易混淆情绪对的辨析

### 3. 回到行动（Action Bridge）

**原则：** 命名之后不能停在"我知道了"，必须给出一个微小但具体的下一步。

**操作要点：**
- 行动必须具体到"现在就能做"：不是"学会调节情绪"，而是"下次感到[情绪]时，先把手放在胸口，做三次深呼吸"
- 用"下次当你感到[情绪]时，试试..."的句式引出行动
- 行动的门槛要极低：30 秒能完成，不需要任何工具
- David 的 tiny tweaks：微小调整比大改变更可持续
- Brackett 的 Meta-Moment：在刺激和反应之间插入一个暂停

**行动类型菜单：**
- 身体层面：深呼吸、把手放在胸口、握拳再松开、站起来走几步
- 认知层面：用一句话写下"我现在感到[精准词]，因为[原因]"
- 社交层面：找一个人说"我现在感觉不太好"（不需要解释为什么）
- 自我关怀层面：允许自己在这个情绪里待一会儿，不急着"好起来"

**理论依据：** David《Emotional Agility》四步法 + Brackett《Permission to Feel》RULER 框架

## 三个账号的 EG 差异化

### 主号（心情可可）

- **受众：** 18-30 岁对情绪敏感的年轻女性
- **核心承诺：** "帮你说清每一种心情"
- **EG 角度：** 广泛的情绪探索，87 种情绪体验都可以覆盖
- **切入特点：** 所有心理学话题都从"你此刻的情绪体验"切入，概念为情绪服务（不是情绪为概念服务）
- **语气：** 温暖、好奇、像朋友聊天
- **常用 EG 框架：**
  - Mood Meter 四象限日常场景
  - "你以为你在 X，其实你在 Y" 系列
  - 情绪 → 需求链条（"你的[情绪]在说：我需要[需求]"）
- **内容比例：** 心理学科普 EG 角度（50%）+ 情绪辨析工具（30%）+ 情绪需求深度（20%）

### 恋爱号（安全感研究所）

- **受众：** 20-30 岁正在恋爱或刚分手的女性
- **核心承诺：** "帮你在恋爱里看清、想清、说清"
- **EG 角度：** 聚焦亲密关系中的情绪辨析（占该号内容约 15%，但 EG 视角贯穿所有内容）
- **切入特点：** 从恋爱冲突场景中的情绪反应入手，而非从依恋理论概念入手
- **语气：** 理解、不评判、帮你看清
- **重点情绪辨析对：**
  - 吃醋 vs 不安全感 — 一个有边界，一个没有
  - 心动 vs 依赖 — 一个是被吸引，一个是离不开
  - 舍不得 vs 不甘心 — 一个关于对方，一个关于自己
  - "不在乎了" 的四种不同情绪 — 释然 / 压抑 / 自我保护 / 绝望
- **与主号的区别：** 主号从"我感觉不好"切入，恋爱号从"我们之间出了什么问题"切入。同一种情绪（如焦虑），主号拆解"你的焦虑到底是什么"，恋爱号拆解"你在这段关系里的焦虑是什么"

### 青少年号（少年心情课）

- **受众：** 12-18 岁青少年 + 35-50 岁家长（双人群号）
- **核心承诺：** "帮你们互相听懂"
- **EG 角度：** 首次命名某种感受的"初体验"——很多青少年不是不想表达，是没有词汇
- **切入特点：** 用学生的场景（考试、友谊、被批评、和家长冲突）和语言做情绪辨析
- **语气（面向青少年）：** 更活泼、用他们的语言、不说教
- **语气（面向家长）：** 平等、不指责、给具体方法
- **重点场景和情绪：**
  - "不想上学" → 拆解为恐惧 / 疲惫 / 无意义感 / 社交回避
  - "无所谓" → 拆解为压抑 / 自我保护 / 情绪枯竭 / 习得性无助
  - "烦死了" → 拆解为愤怒 / 挫败 / 不被理解 / 失控感
  - 被老师当众批评后 → 羞耻 vs 愤怒 vs 委屈（三种不同的反应和不同的需求）
- **家长翻译特色：** 帮家长理解"孩子说 A，听起来像 B，实际在表达 C"
- **安全边界：** 不诊断、不贴标签、不激化亲子矛盾、涉及自伤时明确引导求助

## 内容类型与 EG 融合

### 类型 A：情绪命名帖

**结构：** 场景描述 → 这种感觉叫 XX → 为什么知道名字很重要 → 试试这样做

**适用场景：** 帮用户命名一种常见但模糊的情绪体验

**EG 要点：**
- 场景必须具体到有画面感（时间、地点、身体感觉）
- 命名时给出精准的情绪词，而非心理学术语
- "为什么知道名字很重要"用一句话带过即可（"当你能叫出它的名字，它对你的控制就小了一点"）
- 行动建议来自三板斧的第三步

**适用平台：** 公众号（深度展开）、小红书（精简为卡片）

**理论支撑：** Barrett 身体感觉构建 + Brown 命名即赋权

### 类型 B：情绪辨析帖

**结构：** 你以为是 X，其实是 Y → 区别在哪 → 为什么区分它们很重要 → 对症的处理方式

**适用场景：** 两个容易混淆的情绪之间的辨析

**EG 要点：**
- 先用场景让读者感受到"确实分不清"
- 辨析维度要具体：触发源不同、身体感觉不同、背后需求不同、合适的应对不同
- 不要停在"区分了"就结束，必须给"分清之后怎么办"

**素材来源：** `emotion-differentiation.md` 中的易混淆情绪对

**适用平台：** 公众号、小红书（"你以为你在 X，其实你在 Y" 系列天然适合卡片）

### 类型 C：Mood Meter 互动帖

**结构：** 此刻你在哪个象限？ → 分享你的情绪坐标 → 你想去哪个象限？ → 一个小行动帮你移动

**适用场景：** 轻量互动，建立用户对 Mood Meter 的熟悉度

**EG 要点：**
- 四象限的描述要用日常语言，不用学术语言
- 每个象限给 2-3 个代表性情绪词（来自 `emotion-landscape.md`）
- "移动"的建议要基于 Brackett 的 RULER 框架：Recognize → Understand → Label → Express → Regulate

**适用平台：** 小红书互动（天然适合）、公众号投票

### 类型 D：情绪故事帖

**结构：** 一个具体的故事 → 情绪的命名和转化过程 → 金句收尾

**适用场景：** 用叙事方式展示 EG 的价值，增强情感共鸣

**EG 要点：**
- 故事中必须有"命名"的转折时刻：主人公从模糊的不适到精准的命名
- 转折后要展示命名带来的变化（不一定是情绪变好，可以是"至少知道自己怎么了"）
- 金句收尾不能是鸡汤，要回到 EG 核心——"说清楚就是力量"

**理论支撑：** David 的四步法叙事框架（Showing Up → Stepping Out → Walking Your Why → Moving On）

## 与 topic-evaluation skill 的衔接

### 如何在选题评估中使用 EG

**选题阶段：**
1. 对照 `emotion-landscape.md` 查看目标情绪的位置（哪个象限、有哪些相邻/相关情绪）
2. 用 `emotion-differentiation.md` 检查是否有可挖掘的辨析角度（"这个话题里有没有容易混淆的情绪对"）
3. 确认选题涉及的情绪有理论支撑（Barrett / Brackett / Brown / David 谁的理论最相关）

**角度优化阶段（对应 topic-evaluation Step 2）：**
- topic-evaluation 的 psych-drivers 侧重"读者为什么会点开"（好奇心缺口、自我相关性）
- EG 侧重"用什么情绪体验切入"（身体感觉、场景记忆、精准命名）
- 二者结合：用 psych-drivers 决定标题和钩子，用 EG 决定内容的切入方式和展开逻辑

**标题草拟阶段（对应 topic-evaluation Step 4）：**
- EG 视角的标题倾向于用场景/感受而非概念命名："你说'好的没问题'时胸口那种发紧的感觉"（场景+身体感觉）而非"讨好型人格的5个特征"（概念+列表）
- 参考 topic-evaluation 的 title-templates，但优先选用带有情绪体验元素的模板

### EG 维度的选题评分补充

在 topic-evaluation 的 7 维评分之外，可以额外考虑以下 EG 维度：

| EG 维度 | 评估标准 | 高分示例 | 低分示例 |
|---------|---------|---------|---------|
| 情绪精准度 | 选题是否涉及精准的情绪命名，而非泛泛的情绪大词 | "你的'不甘心'和'委屈'是完全不同的情绪" | "如何管理负面情绪" |
| 辨析价值 | 是否有可辨析的近义/易混淆情绪对 | "焦虑 vs 恐惧：同一种'怕'，不同的需求" | "焦虑的10种表现" |
| 体验可唤起性 | 读者能否在 2 秒内被唤起一个具体的身体/场景记忆 | "深夜刷完手机放下那一刻的空虚" | "手机成瘾的心理机制" |
| 理论深度 | 是否有 Barrett / Brackett / Brown / David 中至少一位的理论支撑 | Barrett 情绪构建论解释"为什么同一件事不同人反应不同" | 纯经验分享，无理论锚点 |

## 附录：EG 视角的自检清单

每篇内容发布前，用以下清单自检：

- [ ] **开头是体验还是概念？** 第一段是否从一个具体的场景/身体感觉开始？
- [ ] **有精准命名吗？** 文中是否出现了至少一个比日常用语更精准的情绪词？
- [ ] **命名有"发现感"吗？** 读者看到这个词后是否会有"原来这种感觉有名字"的反应？
- [ ] **有行动建议吗？** 文末是否给出了一个微小但具体的下一步？
- [ ] **避免说教了吗？** 全文是否用"你有没有过..."而非"你应该..."的语气？
- [ ] **理论隐形了吗？** 心理学理论是否融入了内容（而非单独列出来讲解）？
